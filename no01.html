No.1 標準

確率

目標解答時間：20分

1 問題

袋の中に5枚のカードがあり，それぞれのカードには数字 \(0, 0, 1, 1, 2\) が1つずつ書かれている。この袋から1枚のカードを取り出し，そのカードに書かれている数字を記録してから元へ戻す。この試行を3回繰り返すとき，1回目に記録した数字を \(x\) ，2回目に記録した数字を \(y\) ，3回目に記録した数字を \(z\) とする。
\(N=x+y+z\) として， \(N\) を3で割った余りを \(r\) とする。
  1. (1) \(N=6\) となる確率を求めよ。
  2. (2) \(N=1\) となる確率を求めよ。また，\(r=1\) となる確率を求めよ。
  3. (3) \(r\) の期待値を求めよ。

【条件の整理】

  • 試行の性質：「取り出して元に戻す」を3回繰り返すため、各回の試行は互いに独立である（反復試行）。
  • 1回の確率： 1回の試行で出うる数字の確率は、
    \(0\) が出る確率： \(\frac{2}{5}\)
    \(1\) が出る確率： \(\frac{2}{5}\)
    \(2\) が出る確率： \(\frac{1}{5}\)
  • 確率変数 \(N\) の範囲： \(N=x+y+z\) であり、各文字は \(0, 1, 2\) のいずれかなので、 \(0 \leqq N \leqq 6\) となる。
  • 求めるもの： (1) 特定の \(N\) の確率、(2) \(N=1\) の確率と余り \(r=1\) の確率、(3) 余り \(r\) の期待値（\(r=0, 1, 2\) の確率分布が必要）。

2 解説

問題を解くための考え方から、答案、検算、発展までを順に確認します。

解法選択のトリガー

「元へ戻す」「3回繰り返す」 → 独立な反復試行
1回ごとに各数字が出る確率は変わらないため、それぞれの事象が起こる回数を把握し、その確率の積に「並べ替えの総数（同じものを含む順列）」を掛けて求めます。
「3で割った余り \(r\)」 → とりうる \(N\) の値を余りで分類する
\(N\) は \(0 \leqq N \leqq 6\) の狭い範囲に収まるため、合同式よりも「\(N\) の値を直接列挙して分類」する方が安全で確実です。

思考プロセス

まずは、1回の試行における確率を明確に設定し、これをベースにすべて計算します。

  • Step 1： \(N=6\) は \( (x, y, z) = (2, 2, 2) \) の1通りしかないため、ただちに確率が求まります。
  • Step 2： \(r=1\) となる \(N\) を探します。 \(0 \leqq N \leqq 6\) のうち、3で割って1余る数は \(N=1, 4\) のみです。
  • Step 3： \(N=1\) や \(N=4\) になる \( (x, y, z) \) の組み合わせを見つけ、その「確率の積」と「並べ替えの総数」を掛け合わせます。
  • Step 4： 期待値を求めるには \(r=0, 1, 2\) の確率が必要です。すでに \(P(r=1)\) を求めたので、同様に \(P(r=2)\) を求めます（\(N=2, 5\)）。
  • Step 5： \(P(r=0)\) は \(1 - P(r=1) - P(r=2)\) から出せますが、検算のために直接計算して和が1になるか確かめると完璧です。

実際の答案

1回の試行において、数字 \(0, 1, 2\) のカードを取り出す確率をそれぞれ \(P(0), P(1), P(2)\) とすると、

\[ P(0) = \frac{2}{5}, \quad P(1) = \frac{2}{5}, \quad P(2) = \frac{1}{5} \]

(1) \(N=6\) となる確率

\(x, y, z \in \{0, 1, 2\}\) であり、\(N = x+y+z = 6\) となるのは \((x, y, z) = (2, 2, 2)\) のときのみである。
したがって、求める確率は

\[ \left( \frac{1}{5} \right)^3 = \mathbf{\frac{1}{125}} \]

(2) \(N=1\) となる確率と、\(r=1\) となる確率

\(N=1\) となるのは、\( (x, y, z) \) が \(\{1, 0, 0\}\) の組み合わせのときである。
その並べ方は \(\frac{3!}{2!1!} = 3\) 通りあるため、\(N=1\) となる確率は

\[ P(N=1) = 3 \times P(1) \times \{P(0)\}^2 = 3 \times \frac{2}{5} \times \left(\frac{2}{5}\right)^2 = \mathbf{\frac{24}{125}} \]

次に \(r=1\) となる確率を求める。
\(x, y, z\) のとりうる最大値はそれぞれ2であるから、\(0 \leqq N \leqq 6\) である。
このうち、3で割って余りが1となるのは \(N=1\) または \(N=4\) のときである。
\(N=4\) となる組み合わせは、\(\{2, 2, 0\}\) または \(\{2, 1, 1\}\) である。

  • \(\{2, 2, 0\}\) のとき、並べ方は \(\frac{3!}{2!1!} = 3\) 通り。
    確率は \( 3 \times \left(\frac{1}{5}\right)^2 \times \frac{2}{5} = \frac{6}{125} \)
  • \(\{2, 1, 1\}\) のとき、並べ方は \(\frac{3!}{2!1!} = 3\) 通り。
    確率は \( 3 \times \frac{1}{5} \times \left(\frac{2}{5}\right)^2 = \frac{12}{125} \)

これらは互いに排反であるから、

\[ P(N=4) = \frac{6}{125} + \frac{12}{125} = \frac{18}{125} \]

以上より、\(r=1\) となる確率は

\[ P(r=1) = P(N=1) + P(N=4) = \frac{24}{125} + \frac{18}{125} = \mathbf{\frac{42}{125}} \]

(3) \(r\) の期待値

期待値 \(E\) は \( E = 0 \times P(r=0) + 1 \times P(r=1) + 2 \times P(r=2) \) である。
\(P(r=1)\) は求めたので、\(P(r=2)\) を求める。
\(0 \leqq N \leqq 6\) のうち、3で割って余りが2となるのは \(N=2, 5\) である。

\(N=2\) となる組み合わせは \(\{2, 0, 0\}, \{1, 1, 0\}\) である。

  • \(\{2, 0, 0\}\) の確率：\( 3 \times \frac{1}{5} \times \left(\frac{2}{5}\right)^2 = \frac{12}{125} \)
  • \(\{1, 1, 0\}\) の確率：\( 3 \times \left(\frac{2}{5}\right)^2 \times \frac{2}{5} = \frac{24}{125} \)

よって、\(P(N=2) = \frac{12}{125} + \frac{24}{125} = \frac{36}{125}\)

\(N=5\) となる組み合わせは \(\{2, 2, 1\}\) である。

  • \(\{2, 2, 1\}\) の確率：\( 3 \times \left(\frac{1}{5}\right)^2 \times \frac{2}{5} = \frac{6}{125} \)

よって、\(P(N=5) = \frac{6}{125}\)
したがって、

\[ P(r=2) = P(N=2) + P(N=5) = \frac{36}{125} + \frac{6}{125} = \frac{42}{125} \]

以上より、\(r\) の期待値 \(E\) は

\[ E = 1 \times P(r=1) + 2 \times P(r=2) = 1 \times \frac{42}{125} + 2 \times \frac{42}{125} = \mathbf{\frac{126}{125}} \]

場合分けの整理と確率分布表

求めた確率を整理すると、以下のようになります。

余り \(r\) 和 \(N\) 組み合わせ \(\{x, y, z\}\) 確率
0 0 \(\{0, 0, 0\}\) \(8/125\)
3 \(\{1, 1, 1\}, \{2, 1, 0\}\) \(32/125\)
6 \(\{2, 2, 2\}\) \(1/125\)
1 1 \(\{1, 0, 0\}\) \(24/125\)
4 \(\{2, 2, 0\}, \{2, 1, 1\}\) \(18/125\)
2 2 \(\{2, 0, 0\}, \{1, 1, 0\}\) \(36/125\)
5 \(\{2, 2, 1\}\) \(6/125\)

最終結果と検算

【最終結果】

(1) \(N=6\) となる確率： \(\frac{1}{125}\)

(2) \(N=1\) となる確率： \(\frac{24}{125}\) ， \(r=1\) となる確率： \(\frac{42}{125}\)

(3) \(r\) の期待値： \(\frac{126}{125}\)

【確率の総和による検算】

期待値を求める過程で計算しなかった \(r=0\) の確率を直接求めて、確率の総和が1になるか確認します。
上の表より、\(P(r=0) = \frac{8}{125} + \frac{32}{125} + \frac{1}{125} = \frac{41}{125}\) となります。
\( P(r=0) + P(r=1) + P(r=2) = \frac{41}{125} + \frac{42}{125} + \frac{42}{125} = \frac{125}{125} = 1 \)
総和が1となるため、場合分けの漏れや計算ミスがないことが強く保証されます。

発展：状態推移と行列による一般化

本問は試行が3回でしたが、もし「 \(n\) 回試行したときの和を3で割った余りが \(r\) である確率 \(P_n(r)\) 」を問われたらどうなるでしょうか。この場合、状態推移（漸化式） を用いるのが定石です。

1回カードを引くとき、数字 \(X\) を3で割った余りの確率は、それぞれ
\(X \equiv 0 \pmod 3\) となる確率：\(2/5\)（0を引いたとき）
\(X \equiv 1 \pmod 3\) となる確率：\(2/5\)（1を引いたとき）
\(X \equiv 2 \pmod 3\) となる確率：\(1/5\)（2を引いたとき）

\(n\) 回目までの余りの状態から \(n+1\) 回目の状態がどのように移り変わるかを考えると、以下の連立漸化式が得られます。

\[ \begin{cases} P_{n+1}(0) = \frac{2}{5}P_n(0) + \frac{1}{5}P_n(1) + \frac{2}{5}P_n(2) \\[4pt] P_{n+1}(1) = \frac{2}{5}P_n(1) + \frac{1}{5}P_n(2) + \frac{2}{5}P_n(0) \\[4pt] P_{n+1}(2) = \frac{2}{5}P_n(2) + \frac{1}{5}P_n(0) + \frac{2}{5}P_n(1) \end{cases} \]

行列を用いると次のように美しく表現できます。

\[ \begin{pmatrix} P_{n+1}(0) \\ P_{n+1}(1) \\ P_{n+1}(2) \end{pmatrix} = \frac{1}{5}\begin{pmatrix} 2 & 1 & 2 \\ 2 & 2 & 1 \\ 1 & 2 & 2 \end{pmatrix} \begin{pmatrix} P_{n}(0) \\ P_{n}(1) \\ P_{n}(2) \end{pmatrix} \]

これを \(n=1\) から順番に \(n=3\) まで計算していくと、今回の答えである \((41/125, 42/125, 42/125)\) に完全に一致します。試行回数が多い問題では、この「状態推移」に持ち込む解法が必須となります。

受験生のつまずきポイント

順列（並べ替え）の掛け忘れ

\(N=1\) を求める際、「1, 0, 0」が出るという組み合わせは見つけても、それが「1回目, 2回目, 3回目」のどの順で出るかの考慮を忘れてしまうミスが多発します。\((2/5) \times (2/5)^2\) だけで終わらせず、必ず同じものを含む順列 \(\frac{3!}{2!1!} = 3\) を掛ける必要があります。

\(N\) の最大値の勘違い・場合分けのオーバー

\(N\) を3で割った余りが1になる数を探す際、「\(N=7\) や \(N=10\) はどうだろう？」と考えてしまうことがあります。最初に「最大のカードが2だから、3回引いても最大で \(N=6\) までしかいかない」と変数のとりうる範囲を確定させておかないと、無駄な場合分けに時間を取られてしまいます。

アクティブリコール（復習テスト）

今日の学習内容を思い出すことで、記憶を定着させましょう。質問をクリックして答えを確認してください。

Q1. この問題において、「元へ戻す」「3回繰り返す」という条件から、どのような方針を選びますか？ ▼
毎回同じ確率で起こる「独立な反復試行」として扱います。1回ごとに 0, 1, 2 が出る確率を求め、その確率の積に「並べ替えの総数（順列）」を掛け合わせて計算します。
Q2. \(x,y,z\) の組が \(\{0, 0, 1\}\) である確率は、\(\frac{2}{5} \times \frac{2}{5} \times \frac{2}{5}\) ではありません。答案を作成する際、どのような判断が必要ですか？ ▼
0, 0, 1 が「どの順番で出るか」を考慮する判断が必要です。反復試行では順序が異なるものは別事象となるため、同じものを含む順列を用いて並べ替えの総数である \(\frac{3!}{2!1!} = 3\) 通りを確率に掛ける必要があります。
Q3. 最後に期待値を求める際、計算ミスや場合分けの漏れを防ぐためにどのような検算を行うべきですか？ ▼
余事象を利用して省いた \(r=0\) の確率もあえて直接計算し、「\(r=0, 1, 2\) のすべての確率の総和が \(1\) になるか」を確認すべきです。和が1になれば、場合分けの漏れや重複がないことが強く保証されます。
