1. 問題文と条件整理

1 (確率)

袋の中に5枚のカードがあり、それぞれのカードには数字 0, 0, 1, 1, 2 が1つずつ書かれている。この袋から1枚のカードを取り出し、そのカードに書かれている数字を記録してから元へ戻す。この試行を3回繰り返すとき、1回目に記録した数字を $x$、2回目に記録した数字を $y$、3回目に記録した数字を $z$ とする。

$N = x+y+z$ として、$N$ を3で割った余りを $r$ とする。

  • (1) $N=6$ となる確率を求めよ。
  • (2) $N=1$ となる確率を求めよ。また、$r=1$ となる確率を求めよ。
  • (3) $r$ の期待値を求めよ。
▶ 元の問題画像を確認する

条件の整理

  • カードの構成: $\{0, 0, 1, 1, 2\}$ の計5枚
  • 試行: 1枚引いて元に戻すを3回繰り返す（反復試行・独立）
  • 各数字が出る確率:
    $P(0) = \frac{2}{5}, P(1) = \frac{2}{5}, P(2) = \frac{1}{5}$
  • 変数: $N = x+y+z$
  • $r$ の定義: $N$ を3で割った余り ($r = N \pmod 3$)

最終的な目標

  • (1) $N=6$ となる確率を求める。
  • (2) $N=1$ となる確率、および $r=1$ となる確率を求める。
  • (3) $r$ の期待値 $E[r]$ を求める。

2. 解法選択のトリガー

「元へ戻す」を「3回繰り返す」を見たら

→ 独立試行の確率と判断する。1回ごとの確率は変化しないため、各回の確率の積で事象の確率を計算できる。

「和 $N$ が〇〇になる確率」を見たら

→ 組み合わせ $(x, y, z)$ の列挙と、その並べ替え（順列）の考慮を連想する。$(0,0,1)$ なら $\frac{3!}{2!1!} = 3$ 通りあることを忘れない。

「余り $r$ の期待値」を見たら

→ $r$ がとりうる値は $0, 1, 2$ のみであることに着目し、それぞれの確率 $P(r=0), P(r=1), P(r=2)$ を求めて期待値の定義式 $\sum r_i P_i$ に代入する方針を立てる。

3. 思考プロセス

まず、$x, y, z$ はそれぞれ $0, 1, 2$ のいずれかをとるので、和 $N$ のとりうる最小値は $0+0+0=0$、最大値は $2+2+2=6$ です。

小問の誘導に乗って考えてみましょう。
(1)で $N=6$ となる特殊な場合を求めさせ、(2)で $N=1$ という具体的な和から、条件を拡張して $r=1$（つまり $N=1, 4$）の場合を求めさせています。

この流れから、(3)の期待値計算では、$r=0, 1, 2$ になるための $N$ の値をすべて分類して確率を足し合わせるのが自然な方針（本解）となります。計算量もそれほど多くないので、素直に事象を書き出していくのが一番確実です。

ここから実際の答案

1回の試行において、各数字のカードを取り出す確率はそれぞれ以下の通りである。 $$ P(0) = \frac{2}{5}, \quad P(1) = \frac{2}{5}, \quad P(2) = \frac{1}{5} $$

(1) $N=6$ となる確率

$N = x+y+z = 6$ となるのは、3回とも「2」のカードを取り出した場合のみである。
すなわち、$(x, y, z) = (2, 2, 2)$ の1通り。

したがって、求める確率は

$$ P(N=6) = \left( \frac{1}{5} \right)^3 = \frac{1}{125} $$

答：$ \frac{1}{125} $

(2) $N=1$ となる確率、および $r=1$ となる確率

【前半：$N=1$ となる確率】

$N = x+y+z = 1$ となる数字の組み合わせは $\{0, 0, 1\}$ である。
この順列は $\frac{3!}{2!1!} = 3$ 通りある。

したがって、$N=1$ となる確率は

$$ P(N=1) = 3 \times \left(\frac{2}{5}\right)^2 \times \frac{2}{5} = 3 \times \frac{8}{125} = \frac{24}{125} $$

【後半：$r=1$ となる確率】

$r=1$ となるのは、$N$ を3で割った余りが1のとき、すなわち $N=1, 4$ のいずれかの場合である。（$N$ の最大値は6であるため、$N=7$ 以上は考えない）

$N=4$ となる数字の組み合わせは、$\{0, 2, 2\}$ または $\{1, 1, 2\}$ である。

  • $\{0, 2, 2\}$ の並べ方は $\frac{3!}{2!} = 3$ 通り。
    その確率は $3 \times \frac{2}{5} \times \left(\frac{1}{5}\right)^2 = \frac{6}{125}$
  • $\{1, 1, 2\}$ の並べ方は $\frac{3!}{2!} = 3$ 通り。
    その確率は $3 \times \left(\frac{2}{5}\right)^2 \times \frac{1}{5} = \frac{12}{125}$

よって、$P(N=4) = \frac{6}{125} + \frac{12}{125} = \frac{18}{125}$

これらは互いに排反であるから、

$$ P(r=1) = P(N=1) + P(N=4) = \frac{24}{125} + \frac{18}{125} = \frac{42}{125} $$

答：$P(N=1) = \frac{24}{125}, \quad P(r=1) = \frac{42}{125}$

(3) $r$ の期待値

$r$ のとりうる値は $0, 1, 2$ である。
$P(r=1) = \frac{42}{125}$ はすでに求めたので、残りの $P(r=2)$ を計算する。

$r=2$ となるのは、$N=2, 5$ の場合である。

・$N=2$ となる組み合わせは $\{0, 0, 2\}, \{0, 1, 1\}$

  • $\{0, 0, 2\}$ の並べ方は $3$ 通り。確率は $3 \times \left(\frac{2}{5}\right)^2 \times \frac{1}{5} = \frac{12}{125}$
  • $\{0, 1, 1\}$ の並べ方は $3$ 通り。確率は $3 \times \frac{2}{5} \times \left(\frac{2}{5}\right)^2 = \frac{24}{125}$

よって $P(N=2) = \frac{12}{125} + \frac{24}{125} = \frac{36}{125}$

・$N=5$ となる組み合わせは $\{1, 2, 2\}$

  • $\{1, 2, 2\}$ の並べ方は $3$ 通り。確率は $3 \times \frac{2}{5} \times \left(\frac{1}{5}\right)^2 = \frac{6}{125}$

よって $P(N=5) = \frac{6}{125}$

したがって、$P(r=2) = P(N=2) + P(N=5) = \frac{36}{125} + \frac{6}{125} = \frac{42}{125}$

これらを用いて、$r$ の期待値 $E[r]$ を計算する。（$r=0$ の項は期待値計算で $0$ となるため、確率は明示的に求めなくても計算可能である。）

$$ E[r] = 0 \times P(r=0) + 1 \times P(r=1) + 2 \times P(r=2) $$

$$ E[r] = 0 + 1 \times \frac{42}{125} + 2 \times \frac{42}{125} = \frac{42 + 84}{125} = \frac{126}{125} $$

答：$ \frac{126}{125} $

6. 最終結果と検算

(1) $P(N=6)$ $\frac{1}{125}$
(2) $P(N=1), P(r=1)$ $\frac{24}{125}, \quad \frac{42}{125}$
(3) $E[r]$ $\frac{126}{125}$

【検算：確率の総和は 1 になっているか？】

余事象 $r=0$ となる確率 $P(r=0)$ を求め、全体で 1 になるか確認します。

$N=0, 3, 6$ となる確率をそれぞれ計算します。

  • $P(N=0) = \left(\frac{2}{5}\right)^3 = \frac{8}{125}$
  • $N=3$ の組み合わせは $\{0, 1, 2\}, \{1, 1, 1\}$
    • $\{0, 1, 2\} \rightarrow 6 \times \frac{2}{5} \times \frac{2}{5} \times \frac{1}{5} = \frac{24}{125}$
    • $\{1, 1, 1\} \rightarrow \left(\frac{2}{5}\right)^3 = \frac{8}{125}$
    • $P(N=3) = \frac{24 + 8}{125} = \frac{32}{125}$
  • $P(N=6) = \frac{1}{125}$

$P(r=0) = \frac{8 + 32 + 1}{125} = \frac{41}{125}$

確認：$P(r=0) + P(r=1) + P(r=2) = \frac{41}{125} + \frac{42}{125} + \frac{42}{125} = \frac{125}{125} = 1$

漏れや重複がないことが確認できました。

7. 別解の比較（多項式の展開・母関数）

確率分布を地道に数える代わりに、代数的な展開を利用して一気にすべての確率を求める方法もあります。
1回の試行で引く数字とその確率を対応させた多項式 $f(t) = \frac{2}{5}t^0 + \frac{2}{5}t^1 + \frac{1}{5}t^2$ を考えます。
これを3乗した展開式の $t^k$ の係数が、$N=k$ となる確率に一致します。

$ \left( \frac{2+2t+t^2}{5} \right)^3 = \frac{1}{125} (t^2+2t+2)^3 $

$ (t^2+2t+2)^3 = (t^2+2t+2)(t^4+4t^3+8t^2+8t+4) $
$ = t^6 + 6t^5 + 18t^4 + 32t^3 + 36t^2 + 24t + 8 $

これを見ると、$t^4$ の係数から $P(N=4) = \frac{18}{125}$ などが瞬時に、かつ正確に得られます。
【比較】本解のように組み合わせを書き出すほうが、試験本番では部分点がもらいやすく無難ですが、この母関数的な考え方は見直し・検算ツールとして最強の威力を発揮します。

8. 発展

期待値の線形性を利用するとどうなるか？

$N = x+y+z$ であるから、$N$ の期待値は各回の期待値の和になります。

1回の試行の期待値 $E[x] = 0 \times \frac{2}{5} + 1 \times \frac{2}{5} + 2 \times \frac{1}{5} = \frac{4}{5}$

したがって、$E[N] = E[x] + E[y] + E[z] = \frac{12}{5} = 2.4$ となります。

※ 注意：$N$ の期待値は簡単に求まりますが、$r$ の期待値は $E[N \pmod 3]$ であり、$E[N] \pmod 3$ とは一致しません。 余りや絶対値、2乗などの非線形な操作が入る場合は、本解のように確率分布を求めてから計算する必要があります。

9. 受験生のつまずきポイント

① 同じものを含む順列をかけ忘れる

【誤答】 $\{0, 0, 1\}$ が出る確率は $\frac{2}{5} \times \frac{2}{5} \times \frac{2}{5} = \frac{8}{125}$
【理由】 「1回目に0、2回目に0、3回目に1」という特定の順番しか計算していません。
【防ぐ方法】 組み合わせを見つけたら、常に「並び替えは何通りあるか？」と自問自答し、$\frac{3!}{2!1!} = 3$ を掛ける癖をつけましょう。

② 「3で割った余り」の解釈ミス

【誤答】 $r=1$ となるのは $N=1$ のときだけと考えてしまう。
【理由】 $N$ の最大値が $6$ であることを意識できておらず、$N=4$ の存在を見落としています。
【防ぐ方法】 $N$ の「最小値」と「最大値」を最初に確認し（今回は $0 \le N \le 6$）、その範囲内で条件を満たすものをすべて書き出すようにします。

Q1. 今回のように「カードを戻して複数回引く」問題で、和 $N$ の確率を求めるときの鉄則手順は？ ▼

以下の3ステップを確実に踏む！

  1. $N$ の最小値と最大値を確認する。
  2. 和が $N$ になる「数字の組み合わせ」を全て書き出す。
  3. 各組み合わせについて「並べ替え（順列）」を計算し、各回の確率を掛けて足し合わせる。
Q2. $\{1, 1, 2, 3, 3, 3\}$ という6つの出目から3回選ぶとき、$(1, 3, 3)$ という組み合わせが出る確率はどう計算する？ ▼

まずそれぞれの確率を出します。

$P(1) = \frac{2}{6}=\frac{1}{3}, \quad P(3) = \frac{3}{6}=\frac{1}{2}$

組み合わせ $\{1, 3, 3\}$ の並び順は $\frac{3!}{2!} = 3$ 通りあります。

よって、 $3 \times P(1) \times P(3)^2 = 3 \times \frac{1}{3} \times \left(\frac{1}{2}\right)^2 = \frac{1}{4}$ と計算します。

Q3. 余り $r$ の期待値を求めるとき、$E[N \pmod 3] = E[N] \pmod 3$ のように計算してはいけない理由は？ ▼

期待値の線形性（$E[aX+b] = aE[X]+b$）は成り立ちますが、「剰余（余りを求める操作）」や「2乗」「絶対値」は線形な操作ではないからです。

（例：サイコロ1個の目の期待値は $3.5$ ですが、目を2で割った余りの期待値は $0.5$ であり、$3.5 \pmod 2 = 1.5$ とは一致しません。）

そのため、$r$ ごとの確率を地道に出して $\sum r_i P_i$ で計算する必要があります。
