No.2 基礎

指数関数と方程式

目標解答時間：20分

1 問題

\(x\) の方程式 \(4^x-2^{x+1}+a=0\) (\(a\) は定数) \(\cdots\cdots\) ① がある。
  1. (1) \(2^x=t\) とおくとき，方程式①を \(a, t\) を用いて表せ。
  2. (2) 方程式①が \(x=-1\) を解にもつとき，\(a\) の値を求めよ。このとき，方程式①の \(x=-1\) 以外の解を求めよ。
  3. (3) 方程式①が \(-1 \leqq x \leqq 2\) の範囲に異なる2つの実数解 \(\alpha, \beta\) をもつとき，\(a\) のとり得る値の範囲を求めよ。このとき，\(\alpha+\beta\) のとり得る値の範囲を求めよ。

【条件の整理】

  • 方程式の形： 底が 4 と 2 の指数項が混在している。底を揃えて置換する典型的な形。
  • 実数解の個数条件： (3) では「特定の範囲」に「異なる2つの実数解」をもつ条件が問われているため、2次関数の解の配置（グラフの交点）に帰着させる。
  • 解の対応： 置換した変数 \(t\) と元の変数 \(x\) が1対1に対応するかどうか、また \(x\) の変域から \(t\) の変域がどうなるかを確認する必要がある。
  • 求めるもの： \(a\) の値や範囲に加えて、指数の和 \(\alpha+\beta\) のとり得る範囲を求める。

2 解説

問題を解くための考え方から、答案、検算、発展までを順に確認します。

解法選択のトリガー

「 \(4^x\) と \(2^x\) が混在」 → 共通の塊を作って置換
\(4^x = (2^x)^2\) を利用し、\(2^x = t\) とおいて2次方程式に帰着させます。指数のまま計算を進めることはできません。
「特定の範囲に異なる2つの実数解」 → 2次方程式の解の配置問題へ
\(x\) の方程式の解の条件を、置換した \(t\) の2次方程式の解の条件（軸の位置・頂点の \(y\) 座標・端点の符号）に翻訳します。
「 \(\alpha+\beta\) の値」 → 解と係数の関係の「積」を利用
指数の和を作り出すためには、指数法則 \(2^\alpha \cdot 2^\beta = 2^{\alpha+\beta}\) が必要です。和ではなく積の形を作るのがポイントです。

思考プロセス

  • Step 1： まずは \(t = 2^x\) と置換し、方程式を \(t\) の2次式に書き換えます。
  • Step 2： (3) で一番重要なのは「変域の変換」です。\(x\) の範囲 \(-1 \leqq x \leqq 2\) に対応する \(t\) の範囲を求めます。さらに、\(t=2^x\) において \(x\) と \(t\) が1対1に対応することを確認します。
  • Step 3： 2次関数 \(f(t) = t^2 - 2t + a\) とおき、\(t\) が Step 2 で求めた範囲に異なる2つの実数解をもつための条件を「グラフと \(t\) 軸の交点」として立式します（軸、頂点の \(y\) 座標、端点の符号）。
  • Step 4： 得られた \(a\) の範囲のもとで、\(t_1 t_2 = a\) から \(2^{\alpha+\beta} = a\) を導き、両辺の対数をとって \(\alpha+\beta\) の範囲を求めます。

実際の答案

(1) 方程式の変形

\(4^x = (2^x)^2 = t^2\), \(2^{x+1} = 2 \cdot 2^x = 2t\) であるから、方程式①に代入して

\[ \mathbf{t^2 - 2t + a = 0} \quad \cdots \text{②} \]

(2) \(x=-1\) を解にもつとき

\(x = -1\) のとき、\(t = 2^{-1} = \frac{1}{2}\) である。
これを②に代入すると、

\[ \left(\frac{1}{2}\right)^2 - 2 \cdot \frac{1}{2} + a = 0 \iff \frac{1}{4} - 1 + a = 0 \]

よって、\( \mathbf{a = \frac{3}{4}} \)
このとき、方程式②は

\[ t^2 - 2t + \frac{3}{4} = 0 \iff 4t^2 - 8t + 3 = 0 \]

因数分解して \((2t-1)(2t-3) = 0\) より、\(t = \frac{1}{2}, \frac{3}{2}\)
\(t = \frac{3}{2}\) のとき、\(2^x = \frac{3}{2}\) より

\[ \mathbf{x = \log_2 \frac{3}{2}} \quad (\text{または } \log_2 3 - 1) \]

(3) 実数解の条件と \(\alpha+\beta\) の範囲

\(t=2^x\) は単調増加関数であるから、\(-1 \leqq x \leqq 2\) のとき

\[ 2^{-1} \leqq 2^x \leqq 2^2 \iff \frac{1}{2} \leqq t \leqq 4 \]

また、この範囲において \(x\) と \(t\) は1対1に対応する。
したがって、①が \(-1 \leqq x \leqq 2\) に異なる2つの実数解をもつ条件は、②が \(\frac{1}{2} \leqq t \leqq 4\) に異なる2つの実数解をもつことである。
\(f(t) = t^2 - 2t + a\) とおくと、\(f(t) = (t-1)^2 + a - 1\) であり、
\(y=f(t)\) のグラフは軸が \(t=1\) の下に凸の放物線である。
求める条件は、放物線 \(y=f(t)\) が \(t\) 軸の \(\frac{1}{2} \leqq t \leqq 4\) の部分と異なる2点で交わることであるから、次の3つを同時に満たすことである。

\[ \begin{cases} \text{[1] 軸の条件 } \frac{1}{2} < 1 < 4 & \cdots \text{軸が区間内にある（常に成立）} \\ \text{[2] 頂点の } y \text{ 座標 } f(1) < 0 & \cdots \text{グラフが } t \text{ 軸と交わる} \\ \text{[3] 端点の符号 } f\left(\frac{1}{2}\right) \geqq 0 \text{ かつ } f(4) \geqq 0 & \cdots \text{区間内で交わる} \end{cases} \]
  • [2] について、 \(f(1) = 1^2 - 2 \cdot 1 + a = a - 1 < 0 \iff a < 1\)
  • [3] について、
    \(f\left(\frac{1}{2}\right) = \left(\frac{1}{2}\right)^2 - 2\left(\frac{1}{2}\right) + a = a - \frac{3}{4} \geqq 0 \iff a \geqq \frac{3}{4}\)
    \(f(4) = 4^2 - 2 \cdot 4 + a = a + 8 \geqq 0 \iff a \geqq -8\)
    両方を満たす範囲は \(a \geqq \frac{3}{4}\)

これらを同時に満たす \(a\) の範囲は

\[ \mathbf{\frac{3}{4} \leqq a < 1} \quad \cdots ③ \]

次に、②の2つの解を \(t_1, t_2\) とすると、これらは \(x\) の解 \(\alpha, \beta\) に対応し、\(t_1 = 2^\alpha, t_2 = 2^\beta\) と表せる。
2次方程式②における解と係数の関係より、

\[ t_1 t_2 = a \iff 2^\alpha \cdot 2^\beta = a \iff 2^{\alpha+\beta} = a \]

両辺を底を2とする対数にとると、\(\alpha+\beta = \log_2 a\)
③より \(\frac{3}{4} \leqq a < 1\) であり、底 2 は 1 より大きいから

\[ \log_2 \frac{3}{4} \leqq \log_2 a < \log_2 1 \]

したがって、\(\alpha+\beta\) のとり得る範囲は

\[ \mathbf{\log_2 3 - 2 \leqq \alpha+\beta < 0} \]

図解・可視化

\(y=f(t)\) のグラフが \(1/2 \leqq t \leqq 4\) の範囲で \(t\) 軸と2回交わる様子

最終結果と検算

【最終結果】

(1) \(t^2 - 2t + a = 0\)

(2) \(a = \frac{3}{4}\) , \(x = \log_2 \frac{3}{2}\) (\(\log_2 3 - 1\))

(3) \(\frac{3}{4} \leqq a < 1\) , \(\log_2 3 - 2 \leqq \alpha+\beta < 0\)

【(2) を利用した境界の検算】

(3) で求めた \(a\) の範囲の左端は \(a = 3/4\) です。これは (2) の設定と全く同じです。
(2) のとき、解は \(x = -1\) と \(x = \log_2 3 - 1\) でした。
\(\log_2 3 - 1\) は、\(\log_2 2 < \log_2 3 < \log_2 4\) より \(1 < \log_2 3 < 2\) なので、約 \(0.58\) です。
したがって、2つの解はともに \(-1 \leqq x \leqq 2\) の範囲に収まっており、確かに条件を満たしています。
また、このときの \(\alpha+\beta = -1 + (\log_2 3 - 1) = \log_2 3 - 2\) となり、求めた範囲の左端と完全に一致することが確認でき、自信を持って解答できます。

発展：定数分離による視覚的な解法

方程式 \(t^2 - 2t + a = 0\) の解の配置は、\(a\) を分離して「2つのグラフの交点」として考える方法（定数分離）が非常に有効です。

\[ a = -t^2 + 2t \]

\(y = -t^2 + 2t = -(t-1)^2 + 1\) のグラフ（上に凸の放物線）と、直線 \(y = a\) （水平な直線）の共有点を \(\frac{1}{2} \leqq t \leqq 4\) の範囲で考えます。

  • 頂点：\(t=1\) のとき \(y=1\)
  • 左端：\(t=\frac{1}{2}\) のとき \(y=\frac{3}{4}\)
  • 右端：\(t=4\) のとき \(y=-8\)

横線を上下に動かしたとき、この放物線と異なる2点で交わるためには、横線 \(y=a\) が「頂点 (\(y=1\)) よりわずかに下」から「左端 (\(y=3/4\)) 」までの間を通ればよいことが視覚的にすぐに分かります。
よって、\(\frac{3}{4} \leqq a < 1\) と即答できます。計算ミスを防ぐための強力な武器となります。

受験生のつまずきポイント

変域 \(t\) の範囲を求め忘れる

\(2^x = t\) と置換しただけで満足し、\(x\) の範囲 \(-1 \leqq x \leqq 2\) を無視して「すべての実数」のつもりで頂点の条件 \(f(1)<0\) だけで答えを出してしまうミスが多発します。置換を行ったら、「必ず新しい変数の変域をチェックする」という鉄則を守りましょう。

\(\alpha+\beta\) を求めるために「和」を使ってしまう

「和 \(\alpha+\beta\) が欲しいから、解と係数の関係の和 \(t_1+t_2 = 2\) を使おう」と考えてしまう間違いです。\(t_1+t_2 = 2^\alpha + 2^\beta\) となり、ここから \(\alpha+\beta\) をくくり出すことはできません。指数の和は、元の数の積から生まれる（\(2^\alpha \cdot 2^\beta = 2^{\alpha+\beta}\)）ことをしっかり意識してください。

Q1. 指数関数 \(4^x, 2^x\) が混在する方程式を見たとき、
まず何を考えますか？ ▼
底を 2 に揃え、\(2^x = t\) と置換して \(t\) の2次方程式に帰着させることを考えます。その際、\(x\) の変域から \(t\) の変域（特に指定がなければ \(t>0\)）を必ず求めることが重要です。
Q2. \(x\) の2次方程式ではなく、\(t\) の2次方程式で解の個数を論じる際、
確認しなければならない大前提は何ですか？ ▼
変数 \(x\) と変数 \(t\) が「1対1に対応している」ことです。今回のように \(t=2^x\) の場合、\(t>0\) の範囲において1つの \(t\) に対して1つの \(x\) が定まるため、「\(t\) の解の個数 ＝ \(x\) の解の個数」として扱うことができます。もし \(t = x^2\) のような置換であれば、1つの \(t\) に2つの \(x\) が対応するため注意が必要です。
Q3. \(\alpha+\beta\) の範囲を求める際、
2次方程式の解と係数の関係の「和」ではなく「積」を使ったのはなぜですか？ ▼
2つの解を \(t_1 = 2^\alpha\), \(t_2 = 2^\beta\) としたとき、指数の和 \(\alpha+\beta\) を作り出すためには、指数法則 \(2^\alpha \cdot 2^\beta = 2^{\alpha+\beta}\) を利用する必要があるからです。したがって、\(t_1 + t_2\) ではなく、\(t_1 t_2 = a\) という「積」の関係を用いました。
