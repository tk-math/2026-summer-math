No.4 標準

高次方程式

目標解答時間：20分

1 問題

\(x\) の3次式 \(P(x) = x^3 - (a-1)x^2 + 3(a-2)x - 2a\) がある。ただし，\(a\) は実数の定数とする。
  1. (1) \(P(x)\) を \(x-2\) で割った商を求めよ。
  2. (2) 方程式 \(P(x)=0\) の1つの解が \(1+2i\) であるとき，\(a\) の値を求めよ。ただし，\(i\) は虚数単位とする。
  3. (3) 方程式 \(P(x)=0\) が虚数解をもつとする。このとき，\(P(x)=0\) の3つの解の平方の和が6であるような \(a\) の値を求めよ。

【条件の整理】

  • 実数係数： \(P(x)\) の係数はすべて実数であるため、虚数解をもつ場合は必ず共役な複素数も解になる。
  • (1)の意図： 3次方程式をいきなり解くことは難しいため、因数定理（あるいは割り算）によって1次式と2次式の積に分解する誘導になっている。
  • 解の構造： (1)より、3次方程式 \(P(x)=0\) の解の1つは \(x=2\)（実数）で確定する。残りの2つの解は商の2次方程式から生まれる。
  • 求めるもの： 条件を満たす定数 \(a\) の値。特に(3)では「虚数解をもつ」という前提条件があるため、求めた \(a\) が条件を満たすかどうかの確認（判別式のチェック）が必須となる。

2 解説

問題を解くための考え方から、答案、検算、発展までを順に確認します。

解法選択のトリガー

「(1)の誘導：x-2で割る」 → 因数定理による次数下げ
余りが0になる（\(P(2)=0\)となる）ことが自明な誘導です。割り算を実行して、問題を「1次方程式」と「2次方程式」に分割します。
「実数係数で虚数解1+2i」 → 共役解と解と係数の関係
\(1+2i\) を元の3次式に代入すると計算が大変です。係数が実数なので、共役な複素数 \(1-2i\) も解になります。この2つを解にもつ2次方程式を考え、解と係数の関係を使うのが定石です。
「解の平方の和」 → 対称式の変形 \(\alpha^2+\beta^2 = (\alpha+\beta)^2 - 2\alpha\beta\)
解そのものを解の公式で求めると複雑なルート（または虚数）を含むため、解と係数の関係を利用して和と積の形から平方和を作ります。

思考プロセス

  • Step 1： まずは(1)で \(P(x)\) を \(x-2\) で割り、商である2次式 \(Q(x)\) を求めます。これにより \(P(x) = (x-2)Q(x)\) となります。
  • Step 2： (2)では、\(1+2i\) が解ですが、\(1+2i \neq 2\) であるため、これは \(Q(x) = 0\) の解です。実数係数より \(1-2i\) も解となるので、2次方程式の解と係数の関係を用いて \(a\) を特定します。
  • Step 3： (3)では、まず \(Q(x) = 0\) が虚数解をもつ条件（判別式 \(D<0\)）から \(a\) の範囲を調べます。
  • Step 4： 次に、3つの解（\(2, \alpha, \beta\)）の平方和 \(2^2 + \alpha^2 + \beta^2 = 6\) という等式を立て、\(\alpha^2+\beta^2\) を解と係数の関係で表して \(a\) の2次方程式を解きます。
  • Step 5： 得られた \(a\) の値が Step 3 の判別式条件を満たすか確認し、最終的な答えを絞り込みます。

実際の答案

(1) \(P(x)\) の因数分解

\(P(x)\) を \(x-2\) で割ると、商は \(x^2 - (a-3)x + a\)、余りは \(0\) となる。

2 1 -(a-1) 3(a-2) -2a
2 -2a+6 2a
1 -a+3 a 0

よって、求める商は

\[ \mathbf{x^2 - (a-3)x + a} \]

(2) 虚数解が与えられたときの \(a\) の値

(1)より、\(P(x) = (x-2)(x^2 - (a-3)x + a)\) であるから、方程式 \(P(x)=0\) の解は \(x=2\) と、2次方程式 \(x^2 - (a-3)x + a = 0\) \(\cdots\cdots\) ① の解である。
\(x=1+2i \neq 2\) であるから、\(1+2i\) は①の解である。
①の係数はすべて実数であるから、共役な複素数 \(1-2i\) も①の解となる。
2次方程式の解と係数の関係より、

\[ (1+2i) + (1-2i) = a-3 \implies 2 = a-3 \implies a = 5 \] \[ (1+2i)(1-2i) = a \implies 1^2 - (2i)^2 = 1 - (-4) = 5 \implies a = 5 \]

これらは矛盾しないので、求める \(a\) の値は

\[ \mathbf{a = 5} \]

(3) 平方の和が6となる \(a\) の値

方程式 \(P(x)=0\) が虚数解をもつとき、\(x=2\) は実数であるから、2次方程式①が虚数解をもたなければならない。
①の判別式を \(D\) とすると、\(D < 0\) となる。

\[ D = \{-(a-3)\}^2 - 4 \cdot 1 \cdot a = a^2 - 6a + 9 - 4a = a^2 - 10a + 9 < 0 \] \[ (a-1)(a-9) < 0 \implies 1 < a < 9 \quad \cdots\cdots \text{②} \]

このとき、①の2つの解を \(\alpha, \beta\) とすると、\(P(x)=0\) の3つの解は \(2, \alpha, \beta\) である。
これら3つの解の平方の和が6であるから、

\[ 2^2 + \alpha^2 + \beta^2 = 6 \implies 4 + (\alpha+\beta)^2 - 2\alpha\beta = 6 \]

解と係数の関係より、\(\alpha+\beta = a-3\), \(\alpha\beta = a\) であるから、

\[ 4 + (a-3)^2 - 2a = 6 \] \[ a^2 - 6a + 9 - 2a - 2 = 0 \] \[ a^2 - 8a + 7 = 0 \] \[ (a-1)(a-7) = 0 \] \[ a = 1, 7 \]

このうち、条件② \(1 < a < 9\) を満たすのは \(a=7\) のみである。
したがって、求める \(a\) の値は

\[ \mathbf{a = 7} \]

図解・可視化

3次方程式の解の構造

最終結果と検算

【最終結果】

(1) \(x^2 - (a-3)x + a\)

(2) \(a = 5\)

(3) \(a = 7\)

【代入による検算】

(3)で求めた \(a=7\) が本当に条件を満たすか確認します。
\(a=7\) のとき、2次方程式は \(x^2 - 4x + 7 = 0\) となります。判別式は \(D/4 = (-2)^2 - 7 = -3 < 0\) であり、確かに虚数解をもちます。
この2次方程式の解 \(\alpha, \beta\) は、\(\alpha+\beta = 4\), \(\alpha\beta = 7\) を満たします。
3つの解の平方和は \(2^2 + \alpha^2 + \beta^2 = 4 + (\alpha+\beta)^2 - 2\alpha\beta = 4 + 4^2 - 2 \cdot 7 = 4 + 16 - 14 = 6\) となり、問題文の条件「平方の和が6」と完全に一致します。
一方、不適として除外した \(a=1\) の場合、2次方程式は \(x^2 + 2x + 1 = 0 \iff (x+1)^2=0\) となり、虚数解をもたないため条件違反であることが確認できます。

発展：3次方程式の解と係数の関係による一般化

本問の(3)は、(1)の因数分解を利用して「1次式と2次式」に分けて考えましたが、「3次方程式の解と係数の関係」を知っていれば、因数分解を経由せずに直接計算することができます。

3次方程式 \(Ax^3 + Bx^2 + Cx + D = 0\) の3つの解を \(\alpha, \beta, \gamma\) とすると、

\[ \alpha+\beta+\gamma = -\frac{B}{A}, \quad \alpha\beta+\beta\gamma+\gamma\alpha = \frac{C}{A}, \quad \alpha\beta\gamma = -\frac{D}{A} \]

が成り立ちます。本問の方程式 \(x^3 - (a-1)x^2 + 3(a-2)x - 2a = 0\) の解を \(\gamma_1, \gamma_2, \gamma_3\) とすると、

\[ \gamma_1 + \gamma_2 + \gamma_3 = a-1 \] \[ \gamma_1\gamma_2 + \gamma_2\gamma_3 + \gamma_3\gamma_1 = 3(a-2) \]

3つの解の平方和は、対称式の変形 \((x+y+z)^2 = x^2+y^2+z^2 + 2(xy+yz+zx)\) より、

\[ \gamma_1^2 + \gamma_2^2 + \gamma_3^2 = (\gamma_1+\gamma_2+\gamma_3)^2 - 2(\gamma_1\gamma_2+\gamma_2\gamma_3+\gamma_3\gamma_1) \] \[ \phantom{\gamma_1^2 + \gamma_2^2 + \gamma_3^2} = (a-1)^2 - 2 \cdot 3(a-2) \] \[ \phantom{\gamma_1^2 + \gamma_2^2 + \gamma_3^2} = a^2 - 2a + 1 - 6a + 12 = a^2 - 8a + 13 \]

これが 6 に等しいので、\(a^2 - 8a + 13 = 6 \iff a^2 - 8a + 7 = 0\) となり、答案と同じ2次方程式が得られます。
この知識があれば、もし(1)の因数分解が思いつかなかったり、計算が複雑で割り切れなかったりした場合でも、平方和の条件式までは必ず立式できるようになります。

受験生のつまずきポイント

虚数解をそのまま代入して計算が泥沼化する

(2)で \(x = 1+2i\) を元の3次式 \(P(x)\) や2次式にそのまま代入して計算しようとすると、\((1+2i)^3\) や \((1+2i)^2\) の展開でミスが誘発されます。実数係数という条件を活かし、「共役な複素数も解であること」と「解と係数の関係」を組み合わせるのが最もスマートで確実な方針です。

「虚数解をもつ」という前提条件の確認を忘れる

(3)で \(a^2 - 8a + 7 = 0\) を解いて \(a=1, 7\) と出た段階で満足し、そのまま両方を答えにしてしまうミスが非常に多いです。問題文の冒頭に「虚数解をもつとする」という前提があるため、商の2次方程式の判別式 \(D < 0\) を用いて \(a\) の範囲を絞り込まないと、減点対象になります。出てきた値は必ず条件と照らし合わせる癖をつけましょう。

アクティブリコール（復習テスト）

今日の学習内容を思い出すことで、記憶を定着させましょう。質問をクリックして答えを確認してください。

Q1. 実数係数のn次方程式が虚数 \(p+qi\) を解にもつとき、
他にどのような性質を持つ解が存在すると言えますか？
▼
共役な複素数である \(p-qi\) も必ず解にもちます。これを利用することで、その2つの虚数解をもつ2次方程式を「和と積（解と係数の関係）」から簡単に復元できます。
Q2. 2次方程式の解 \(\alpha, \beta\) の平方の和 \(\alpha^2+\beta^2\) を、
解と係数の関係を用いて式で表すにはどう変形しますか？
▼
対称式の基本変形である \(\alpha^2+\beta^2 = (\alpha+\beta)^2 - 2\alpha\beta\) を用います。これにより、解の和と積の形になり、係数だけで値を計算できるようになります。
Q3. (3)で平方和の条件から \(a\) の値を求めた後、
それが最終的な答えになる前に必ず確認すべき条件は何ですか？
▼
「方程式が虚数解をもつ」という前提条件です。2次方程式の判別式 \(D < 0\) を満たすか（本問では \(1 < a < 9\)）を確認し、条件を満たさない \(a\) の値（本問では \(a=1\)）を除外する必要があります。
