No.5 標準

三角関数

目標解答時間：20分

1 問題

\(P = \sin\left(x+\frac{\pi}{3}\right)\), \(Q = \cos\left(x+\frac{\pi}{6}\right)\) とする。
  1. (1) \(x = \frac{\pi}{2}\) のとき、\(P\) の値を求めよ。
  2. (2) \(P\) を \(a\sin x + b\cos x\) の形で表せ。ただし、\(a, b\) は定数とする。また、\(x\) についての方程式 \(P+Q = \frac{3}{2}\) を \(0 \leqq x \leqq \pi\) の範囲で解け。
  3. (3) \(x\) についての関数 \(y = PQ+P+Q\) の \(0 \leqq x \leqq \pi\) における最小値とそのときの \(x\) の値を求めよ。

【条件の整理】

  • 異なる位相の三角関数： \(P\) と \(Q\) は位相（角度部分）が異なっているため、そのままでは足し引きが難しい。
  • 関数の統一： (2) の誘導により加法定理を用いて展開し、角をすべて \(x\) に統一することが方針となる。
  • 変域の制限： \(x\) の範囲は \(0 \leqq x \leqq \pi\) に制限されている。方程式の解や関数の最小値を求める際、この範囲を厳密に考慮する必要がある。
  • 求めるもの： (2) では三角関数の合成（または直接計算）による方程式の解。(3) では対称式のような形 \(PQ+P+Q\) の最小値。

2 解説

問題を解くための考え方から、答案、検算、発展までを順に確認します。

解法選択のトリガー

「異なる角の三角関数の和や積」 → 加法定理で角を揃える
\(x+\frac{\pi}{3}\) と \(x+\frac{\pi}{6}\) のように角がずれている場合、まずは加法定理を用いて角を \(x\) のみに統一するのが三角関数の基本手筋です。
「\(\cos x\) や \(\sin x\) の2次式の最大・最小」 → \(t\) とおいて2次関数に帰着
角を揃えた後、\(\cos x\) または \(\sin x\) だけで式を表現できる場合は、1つの文字（例えば \(t = \cos x\)）に置き換えて平方完成を行います。その際、必ず \(t\) の変域を確認します。

思考プロセス

  • Step 1： (1) は単純に代入して計算します。
  • Step 2： (2) は加法定理を用いて \(P\) と \(Q\) を展開します。すると都合よく \(\sin x\) の項が消去され、\(P+Q\) が \(\cos x\) だけで表せます。これにより方程式が簡単に解けます。
  • Step 3： (3) の \(PQ\) を計算します。(2) で得た形を見ると、和と差の積 \((A+B)(A-B)\) の形になっているため、簡単に展開できます。これを \(\cos^2 x\) だけの式に整理します。
  • Step 4： 関数 \(y\) を \(\cos x\) の2次関数として表し、\(t = \cos x\) と置換します。定義域 \(0 \leqq x \leqq \pi\) から、\(t\) の変域を \(-1 \leqq t \leqq 1\) と決定します。
  • Step 5： 平方完成を行い、定義域内における頂点の位置と端点の値を確認して最小値を求めます。最後に \(t\) から \(x\) の値へ戻します。

実際の答案

(1) 代入計算

\(x = \frac{\pi}{2}\) を \(P = \sin\left(x+\frac{\pi}{3}\right)\) に代入すると、

\[ P = \sin\left(\frac{\pi}{2} + \frac{\pi}{3}\right) = \sin\left(\frac{5\pi}{6}\right) = \mathbf{\frac{1}{2}} \]

(2) \(P\) の変形と方程式の解

加法定理を用いて \(P, Q\) を展開すると、

\[ P = \sin\left(x+\frac{\pi}{3}\right) = \sin x \cos\frac{\pi}{3} + \cos x \sin\frac{\pi}{3} = \mathbf{\frac{1}{2}\sin x + \frac{\sqrt{3}}{2}\cos x} \]
\[ Q = \cos\left(x+\frac{\pi}{6}\right) = \cos x \cos\frac{\pi}{6} - \sin x \sin\frac{\pi}{6} = \frac{\sqrt{3}}{2}\cos x - \frac{1}{2}\sin x \]

よって、\(P+Q\) は

\[ P+Q = \left(\frac{1}{2} - \frac{1}{2}\right)\sin x + \left(\frac{\sqrt{3}}{2} + \frac{\sqrt{3}}{2}\right)\cos x = \sqrt{3}\cos x \]

方程式 \(P+Q = \frac{3}{2}\) は、

\[ \sqrt{3}\cos x = \frac{3}{2} \iff \cos x = \frac{3}{2\sqrt{3}} \iff \cos x = \frac{\sqrt{3}}{2} \]

\(0 \leqq x \leqq \pi\) の範囲でこれを解くと、

\[ \mathbf{x = \frac{\pi}{6}} \]

(3) 関数の最小値

\(y = PQ+P+Q\) について、まず \(PQ\) を計算する。(2) の結果より、

\[ PQ = \left(\frac{\sqrt{3}}{2}\cos x + \frac{1}{2}\sin x\right)\left(\frac{\sqrt{3}}{2}\cos x - \frac{1}{2}\sin x\right) \] \[ \phantom{PQ} = \frac{3}{4}\cos^2 x - \frac{1}{4}\sin^2 x \]

\(\sin^2 x = 1 - \cos^2 x\) を代入して、

\[ PQ = \frac{3}{4}\cos^2 x - \frac{1}{4}(1 - \cos^2 x) = \frac{3}{4}\cos^2 x - \frac{1}{4} + \frac{1}{4}\cos^2 x = \cos^2 x - \frac{1}{4} \]

よって、関数 \(y\) は

\[ y = \left(\cos^2 x - \frac{1}{4}\right) + \sqrt{3}\cos x = \cos^2 x + \sqrt{3}\cos x - \frac{1}{4} \]

ここで、\(t = \cos x\) とおく。\(0 \leqq x \leqq \pi\) より、\(t\) のとりうる値の範囲は

\[ -1 \leqq t \leqq 1 \quad \cdots\cdots \text{①} \]

\(y\) を \(t\) の関数として表し、平方完成すると

\[ y = t^2 + \sqrt{3}t - \frac{1}{4} = \left(t + \frac{\sqrt{3}}{2}\right)^2 - \frac{3}{4} - \frac{1}{4} = \left(t + \frac{\sqrt{3}}{2}\right)^2 - 1 \]

このグラフは下に凸の放物線であり、頂点は \(\left(-\frac{\sqrt{3}}{2}, -1\right)\) である。
\(-\frac{\sqrt{3}}{2} \fallingdotseq -0.866\) であるから、頂点の \(t\) 座標は範囲①に含まれる。
したがって、\(y\) は \(t = -\frac{\sqrt{3}}{2}\) のとき、最小値 \(-1\) をとる。
このとき、\(\cos x = -\frac{\sqrt{3}}{2}\) であり、\(0 \leqq x \leqq \pi\) より \(x = \frac{5\pi}{6}\) である。
以上より、求める最小値と \(x\) の値は、

\[ \mathbf{x = \frac{5\pi}{6} \text{ のとき、最小値 } -1} \]

グラフによる確認

\(y = \left(t + \frac{\sqrt{3}}{2}\right)^2 - 1\) のグラフと定義域 (\(-1 \leqq t \leqq 1\))

最終結果と検算

【最終結果】

(1) \( \frac{1}{2} \)

(2) \(P = \frac{1}{2}\sin x + \frac{\sqrt{3}}{2}\cos x \), \quad \(x = \frac{\pi}{6}\)

(3) \(x = \frac{5\pi}{6}\) のとき、最小値 \(-1\)

【代入による直接検算】

(3) で求めた \(x = \frac{5\pi}{6}\) を、問題文の最初の式に直接代入して矛盾がないか確かめます。
\( P = \sin\left(\frac{5\pi}{6} + \frac{\pi}{3}\right) = \sin\left(\frac{7\pi}{6}\right) = -\frac{1}{2} \)
\( Q = \cos\left(\frac{5\pi}{6} + \frac{\pi}{6}\right) = \cos(\pi) = -1 \)
このとき、\( y = PQ+P+Q \) に代入すると、
\( y = \left(-\frac{1}{2}\right) \times (-1) + \left(-\frac{1}{2}\right) + (-1) = \frac{1}{2} - \frac{1}{2} - 1 = -1 \)
求めた最小値 \(-1\) と完全に一致したため、加法定理や平方完成の計算がすべて正しかったことが保証されます。

発展：和積・積和の公式によるショートカット

本問は「加法定理」で展開するのが標準的ですが、\(Q\) を少し変形すると、和積の公式・積和の公式をダイレクトに使って計算量を減らすことができます。
\(\cos \theta = \sin\left(\frac{\pi}{2} - \theta\right)\) という性質を利用して、\(Q\) を \(\sin\) に変換します。

\[ Q = \cos\left(x+\frac{\pi}{6}\right) = \sin\left\{\frac{\pi}{2} - \left(x+\frac{\pi}{6}\right)\right\} = \sin\left(\frac{\pi}{3} - x\right) \]

(2)の和の計算

和積の公式 \(\sin A + \sin B = 2\sin\frac{A+B}{2}\cos\frac{A-B}{2}\) を用います。

\[ P+Q = \sin\left(x+\frac{\pi}{3}\right) + \sin\left(\frac{\pi}{3} - x\right) = 2\sin\left(\frac{\frac{\pi}{3}+\frac{\pi}{3}}{2}\right)\cos\left(\frac{2x}{2}\right) = 2\sin\frac{\pi}{3}\cos x = \sqrt{3}\cos x \]

(3)の積の計算

積和の公式 \(\sin A \sin B = -\frac{1}{2}\{\cos(A+B) - \cos(A-B)\}\) を用います。

\[ PQ = \sin\left(x+\frac{\pi}{3}\right)\sin\left(\frac{\pi}{3} - x\right) = -\frac{1}{2}\left\{\cos\left(\frac{2\pi}{3}\right) - \cos 2x\right\} = -\frac{1}{2}\left(-\frac{1}{2} - \cos 2x\right) = \frac{1}{4} + \frac{1}{2}\cos 2x \]

2倍角の公式 \(\cos 2x = 2\cos^2 x - 1\) を代入すると、

\[ PQ = \frac{1}{4} + \frac{1}{2}(2\cos^2 x - 1) = \cos^2 x - \frac{1}{4} \]

このように、高度な公式を使いこなせれば、展開の煩雑な計算や \(\sin^2 x = 1-\cos^2 x\) の変換を一気に飛び越えることができます。

受験生のつまずきポイント

置換した変数の変域 \(-1 \leqq t \leqq 1\) を忘れる

\(t = \cos x\) と置換した直後に、変域を確認せずに平方完成を進めてしまうミスが非常に多いです。本問ではたまたま頂点（\(t = -\sqrt{3}/2\)）が範囲内に収まっていたため答えは合いますが、もし頂点が範囲外にあった場合は、端点で最小値をとることになります。変域の記述がない答案は大幅な減点対象になります。

加法定理の符号や値の思い違い

\(Q = \cos\left(x+\frac{\pi}{6}\right)\) を展開する際、\(\cos(\alpha+\beta) = \cos\alpha\cos\beta - \sin\alpha\sin\beta\) となるべきところを、うっかりプラスにしてしまう（\(\cos\alpha\cos\beta + \sin\alpha\sin\beta\)）ミスが多発します。加法定理の符号は必ず確認しながら計算を進めましょう。

Q1. 角度（位相）が異なる三角関数の式が混在しているとき、
最初に取るべき定石のアプローチは何ですか？
▼
加法定理を用いて展開し、角をすべて同じもの（今回は \(x\) ）に揃えることです。角が揃って初めて、同類項をまとめたり、一つの変数に置き換えたりすることが可能になります。
Q2. \(\cos x\) を \(t\) に置き換えて2次関数の最大・最小を考える際、
絶対に忘れてはならない確認事項は何ですか？
▼
置き換えた文字 \(t\) の「とりうる値の範囲（変域）」です。\(0 \leqq x \leqq \pi\) の場合、\(t\) の範囲は \(-1 \leqq t \leqq 1\) に制限され、この範囲の中でグラフの頂点や端点を調べる必要があります。
Q3. (3)で求めた「\(x = \frac{5\pi}{6}\) のとき最小値 \(-1\)」という結果は、
どのようにして確実な検算ができますか？
▼
求めた \(x = \frac{5\pi}{6}\) を、問題文の最初の式 \(P=\sin(x+\frac{\pi}{3})\) と \(Q=\cos(x+\frac{\pi}{6})\) に直接代入し、得られた \(P\) と \(Q\) の値から \(y=PQ+P+Q\) を計算して、最小値 \(-1\) と一致するかを確かめる方法です。
