No.6 標準

ベクトル

目標解答時間：25分

1 問題

\(\mathrm{OA}=3, \mathrm{OB}=2, \angle\mathrm{AOB}=60^\circ\) の平行四辺形 \(\mathrm{OACB}\) がある。辺 \(\mathrm{BC}\) の中点を \(\mathrm{D}\) とし，辺 \(\mathrm{AC}\) 上に点 \(\mathrm{E}\) を \(\vec{\mathrm{OE}} = \vec{\mathrm{OA}} + t\vec{\mathrm{AC}}\) (\(0
  • (1) \(\vec{\mathrm{OC}}\), \(\vec{\mathrm{OD}}\) をそれぞれ \(\vec{a}\), \(\vec{b}\) を用いて表せ。また，内積 \(\vec{a}\cdot\vec{b}\) の値を求めよ。
  • (2) \(\vec{\mathrm{OE}}\) を \(t, \vec{a}, \vec{b}\) を用いて表せ。また，\(\vec{\mathrm{OE}} \perp \vec{\mathrm{AD}}\) のとき，\(t\) の値を求めよ。
  • (3) (2)のとき，線分 \(\mathrm{OE}\) と線分 \(\mathrm{AD}\) の交点を \(\mathrm{F}\) とする。\(\vec{\mathrm{OF}}\) を \(\vec{a}, \vec{b}\) を用いて表せ。また，\(\triangle\mathrm{OAF}\) の面積を \(S_1\)，\(\triangle\mathrm{DEF}\) の面積を \(S_2\) とするとき，\(S_1:S_2\) を最も簡単な整数の比で表せ。

    • 【条件の整理】

    • ベクトルの設定： 基底 \(\vec{a}, \vec{b}\) が与えられている。 \(\vec{a}\cdot\vec{a} = 9, \vec{b}\cdot\vec{b} = 4, \vec{a}\cdot\vec{b} = 3 \times 2 \times \cos 60^\circ = 3\)。
    • 位置関係： \(\vec{\mathrm{OC}} = \vec{a}+\vec{b}\), \(\vec{\mathrm{OD}} = \vec{b} + \frac{1}{2}\vec{a}\), \(\vec{\mathrm{OE}} = \vec{a} + t\vec{b}\) (平行四辺形より \(\vec{\mathrm{AC}}=\vec{\mathrm{OB}}=\vec{b}\))。
    • 求めるもの： \(\vec{\mathrm{OE}} \perp \vec{\mathrm{AD}}\) より内積 0 を利用。交点 \(\mathrm{F}\) は比 \(s : 1-s\) で表現。面積比は、線分 \(\mathrm{AD}\) を底辺のある直線と見て、底辺の比と高さの比で求める。

    2 解説

    問題を解くための考え方から、答案、検算、発展までを順に確認します。

    解法選択のトリガー

    「垂直条件」 → 内積 = 0
    2つのベクトルが垂直であれば、それらの内積が 0 になります。未知数 \(t\) を含んだ式を \(\vec{a}\cdot\vec{b}\) の値を使って整理します。
    「交点 \(\mathrm{F}\) 」 → 2通りに表して係数比較
    FはOE上の点かつAD上の点なので、\(\vec{\mathrm{OF}} = s\vec{\mathrm{OE}}\) かつ \(\vec{\mathrm{OF}} = (1-k)\vec{\mathrm{OA}} + k\vec{\mathrm{OD}}\) 等として表現します。

    思考プロセス

    • Step 1： \(\vec{a}, \vec{b}\) の内積を計算し、平行四辺形の定義に基づいて各頂点ベクトルを表現する。
    • Step 2： \(\vec{\mathrm{OE}} \perp \vec{\mathrm{AD}}\) より \(\vec{\mathrm{OE}}\cdot\vec{\mathrm{AD}} = 0\) を計算し、\(t\) を求める。
    • Step 3： 交点 \(\mathrm{F}\) をベクトル方程式を用いて求め、\(\vec{\mathrm{OF}}\) を確定させる。
    • Step 4： 面積比 \(S_1:S_2\) を、\(\mathrm{AD}\) 上の底辺の比と、\(\mathrm{OE} \perp \mathrm{AD}\) から分かる高さの比で算出する。

    実際の答案

    (1) ベクトルの表現と内積

    \[ \vec{\mathrm{OC}} = \vec{a} + \vec{b}, \quad \vec{\mathrm{OD}} = \vec{b} + \frac{1}{2}\vec{a} \] \[ \vec{a}\cdot\vec{b} = |\vec{a}||\vec{b}|\cos 60^\circ = 3 \cdot 2 \cdot \frac{1}{2} = 3 \]

    (2) \(t\) の決定

    \[ \vec{\mathrm{OE}} = \vec{a} + t\vec{b}, \quad \vec{\mathrm{AD}} = \vec{\mathrm{OD}} - \vec{\mathrm{OA}} = \frac{1}{2}\vec{a} + \vec{b} - \vec{a} = -\frac{1}{2}\vec{a} + \vec{b} \] \[ \vec{\mathrm{OE}} \cdot \vec{\mathrm{AD}} = (\vec{a} + t\vec{b}) \cdot \left(-\frac{1}{2}\vec{a} + \vec{b}\right) = -\frac{1}{2}|\vec{a}|^2 + \vec{a}\cdot\vec{b} - \frac{1}{2}t\vec{a}\cdot\vec{b} + t|\vec{b}|^2 \] \[ = -\frac{1}{2} \cdot 9 + 3 - \frac{3}{2}t + 4t = -\frac{9}{2} + 3 + \frac{5}{2}t = \frac{5}{2}t - \frac{3}{2} \] \[ \frac{5}{2}t - \frac{3}{2} = 0 \implies \mathbf{t = \frac{3}{5}} \]

    (3) \(\vec{\mathrm{OF}}\) と面積比

    \(\mathrm{F}\) は線分 \(\mathrm{OE}\) 上の点なので \(\vec{\mathrm{OF}} = s\vec{\mathrm{OE}} = s\left(\vec{a} + \frac{3}{5}\vec{b}\right) = s\vec{a} + \frac{3}{5}s\vec{b}\)。
    また \(\mathrm{F}\) は線分 \(\mathrm{AD}\) 上なので \(\vec{\mathrm{OF}} = (1-k)\vec{\mathrm{OA}} + k\vec{\mathrm{OD}} = (1-k)\vec{a} + k\left(\frac{1}{2}\vec{a} + \vec{b}\right) = \left(1-\frac{1}{2}k\right)\vec{a} + k\vec{b}\)。
    係数を比較して \(s = 1-\frac{1}{2}k\), \(\frac{3}{5}s = k\)。代入して \(s = 1 - \frac{1}{2}\left(\frac{3}{5}s\right) = 1 - \frac{3}{10}s \implies \frac{13}{10}s = 1 \implies s = \frac{10}{13}\)。

    \[ \vec{\mathrm{OF}} = \frac{10}{13}\vec{a} + \frac{6}{13}\vec{b} \]

    ここからは、底辺と高さの比だけで面積比を求める。
    まず、\(\mathrm{F}\) は \(\mathrm{AD}\) 上の点で、上の係数比較より \(k=\frac{6}{13}\) である。したがって、

    \[ \mathrm{AF}:\mathrm{FD}=\frac{6}{13}:\frac{7}{13}=6:7 \]

    また、\(\mathrm{F}\) は \(\mathrm{OE}\) 上の点で、\(\vec{\mathrm{OF}}=\frac{10}{13}\vec{\mathrm{OE}}\) だから、

    \[ \mathrm{OF}:\mathrm{FE}=\frac{10}{13}:\frac{3}{13}=10:3 \]

    さらに、\(\mathrm{OE}\perp\mathrm{AD}\) なので、\(\triangle\mathrm{OAF}\) の高さは \(\mathrm{OF}\)、\(\triangle\mathrm{DEF}\) の高さは \(\mathrm{FE}\) である。よって、

    \[ S_1:S_2=(\mathrm{AF}\times\mathrm{OF}):(\mathrm{DF}\times\mathrm{FE})=(6\times10):(7\times3)=60:21=20:7 \]

    したがって、面積比は \(\boxed{S_1:S_2=20:7}\)。

    図解・可視化

    最終結果と検算

    最終結果: (1) \(\vec{\mathrm{OC}} = \vec{a}+\vec{b}, \vec{\mathrm{OD}} = \frac{1}{2}\vec{a}+\vec{b}, \vec{a}\cdot\vec{b} = 3\), (2) \(\vec{\mathrm{OE}} = \vec{a} + t\vec{b}, t=\frac{3}{5}\), (3) \(\vec{\mathrm{OF}} = \frac{10}{13}\vec{a}+\frac{6}{13}\vec{b}, S_1:S_2 = 20:7\)

    発展：面積比の見方

    面積比を求める際は、\(\triangle \mathrm{OAF}\) と \(\triangle \mathrm{DEF}\) の底辺をどちらも直線 \(\mathrm{AD}\) 上に置くと見通しがよくなります。\(\mathrm{OE}\perp\mathrm{AD}\) なので、高さはそれぞれ \(\mathrm{OF}\), \(\mathrm{FE}\) です。したがって、底辺の比 \(\mathrm{AF}:\mathrm{FD}\) と高さの比 \(\mathrm{OF}:\mathrm{FE}\) を掛ければ、面積比が求まります。

    受験生のつまずきポイント

    内積の計算ミス

    内積 \(\vec{a}\cdot\vec{b} = 3\) を \(\vec{a}\cdot\vec{b} = 6\) と間違えるミス。角度が \(60^\circ\) なので \(\cos 60^\circ = 1/2\) を掛けることを忘れないように。

    Q1. 2つのベクトルが垂直である条件を、
    ベクトルを使ってどのように記述しますか？     ▼
    2つのベクトル \(\vec{u}, \vec{v}\) が垂直である条件は、内積 \(\vec{u} \cdot \vec{v} = 0\) です。
    Q2. 線分上の交点 \(\mathrm{F}\) を求める際、
    どのような手法が最も確実ですか？     ▼
    交点 \(\mathrm{F}\) を2通りの線分上の点としてベクトルで表し、一次独立な2つのベクトル \(\vec{a}, \vec{b}\) の係数を比較して連立方程式を解く方法です。
    Q3. 三角形の面積比を求める際、
    底辺と高さに注目する方法は何ですか？     ▼
    2つの三角形の底辺を同じ直線上に取り、面積比を「底辺の比 × 高さの比」で求めます。この問題では、底辺の比は \(\mathrm{AF}:\mathrm{FD}=6:7\)、高さの比は \(\mathrm{OF}:\mathrm{FE}=10:3\) なので、\(S_1:S_2=(6\times10):(7\times3)=20:7\) となります。
