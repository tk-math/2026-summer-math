No.9 基礎

三角関数

目標解答時間：20分

1 問題

\(0 \le x < 2\pi\) で定義された\(x\)の関数 \(y=\dfrac{3}{2}\cos 2x - 4\sin x + \dfrac{3}{2}\) がある。

  1. \(x=\dfrac{\pi}{2}\) のとき，\(y\)の値を求めよ。
  2. \(t=\sin x\) とおく。\(0 \le x < 2\pi\) のとき，\(t\)のとり得る値の範囲を求めよ。また，\(y\)を\(t\)の式で表せ。
  3. 関数\(y\)の値域を求めよ。また，\(x\)の方程式 \(y-4=0\) の解の個数を求めよ。

【条件の整理】

  • 関数式： \(\cos 2x\) と \(\sin x\) が混在している。種類（\(\sin, \cos\)）と角（\(x, 2x\)）を統一することが第一歩。
  • 定義域： \(0 \le x < 2\pi\)。この範囲内で \(\sin x\) がとる値の範囲を考える。
  • 変数の置き換え： \(t = \sin x\)。文字を置き換えたら、必ず新しい文字の定義域（範囲）を確認する。
  • 解の個数の対応： \(t\) の値1つにつき、\(x\) が何個存在するかは \(t\) の値によって異なる。単位円上で \(\sin x = t\) となる交点の数を考える。

2 解説

問題を解くための考え方から、答案、検算、発展までを順に確認します。

解法選択のトリガー

角と関数がバラバラな式を見たら

「角の統一」と「関数の統一」を行います。本問では \(2x\) を \(x\) に直すため、2倍角の公式を使います。さらに、式の中に \(\sin x\) があるため、\(\cos 2x = 1 - 2\sin^2 x\) を選択し、全体を \(\sin x\) だけの式に統一します。

文字の置き換え（\(t = \sin x\)）を行ったら

何かを別の文字で置いたら、必ずその新しい文字の「変域（とり得る値の範囲）」を明記します。これを忘れると、最大値・最小値や解の個数を間違える原因になります。

三角方程式の解の個数と来たら

置き換えた文字 \(t\) の個数と、元の文字 \(x\) の個数は必ずしも「1対1」ではありません。\(t=\sin x\) の場合、\(t=\pm 1\) のときは \(x\) は1個ですが、\(-1 < t < 1\) のときは \(x\) は2個存在します。単位円を描いて視覚的に確認する習慣をつけましょう。

思考プロセス

(1)の計算方針
直接 \(x = \dfrac{\pi}{2}\) を代入して計算します。\(\cos\left(2 \cdot \dfrac{\pi}{2}\right) = \cos \pi\) となることに注意しましょう。
(2)の変形方針
\(0 \le x < 2\pi\) なので、\(\sin x\) のグラフや単位円を思い浮かべれば、\(t\) の範囲は \(-1 \le t \le 1\) とすぐにわかります。
次に式変形は \(\cos 2x = 1 - 2\sin^2 x = 1 - 2t^2\) を用いて代入し、\(t\) の2次関数として整理します。
(3)の値域と解の個数の数え方
(2)で得た \(y\) (\(t\) の2次関数) の値域を、定義域 \(-1 \le t \le 1\) の下で求めます。平方完成して頂点と端点の値を確認します。
方程式 \(y-4=0\) すなわち \(y=4\) の解の個数は、まず2次方程式を解いて \(t\) の値を求めます。得られた \(t\) の値それぞれについて、\(-1 \le t \le 1\) の範囲に入っているかを確認し、入っている場合はその \(t\) に対する \(x\) の個数をカウントして合計します。

実際の答案

（1）

\(x=\dfrac{\pi}{2}\) のとき，

\begin{aligned} y &= \frac{3}{2}\cos\left(2 \cdot \frac{\pi}{2}\right) - 4\sin\frac{\pi}{2} + \frac{3}{2} \\ &= \frac{3}{2}\cos\pi - 4\sin\frac{\pi}{2} + \frac{3}{2} \\ &= \frac{3}{2}(-1) - 4(1) + \frac{3}{2} \\ &= -\frac{3}{2} - 4 + \frac{3}{2} \\ &= -4 \end{aligned}
答: \(y = -4\)

（2）

\(0 \le x < 2\pi\) において，\(\sin x\) のとり得る値の範囲は \(-1 \le \sin x \le 1\) である。
\(t = \sin x\) より，\(t\) のとり得る値の範囲は

\[ -1 \le t \le 1 \]

2倍角の公式より \(\cos 2x = 1 - 2\sin^2 x = 1 - 2t^2\) であるから，\(y\) を \(t\) で表すと

\begin{aligned} y &= \frac{3}{2}(1 - 2t^2) - 4t + \frac{3}{2} \\ &= \frac{3}{2} - 3t^2 - 4t + \frac{3}{2} \\ &= -3t^2 - 4t + 3 \end{aligned}
答: \(t\)の範囲は \(-1 \le t \le 1\)，式は \(y = -3t^2 - 4t + 3\)

（3）

\(y\) の式を平方完成すると，

\begin{aligned} y &= -3\left(t^2 + \frac{4}{3}t\right) + 3 \\ &= -3\left(t + \frac{2}{3}\right)^2 + 3\left(\frac{2}{3}\right)^2 + 3 \\ &= -3\left(t + \frac{2}{3}\right)^2 + \frac{4}{3} + \frac{9}{3} \\ &= -3\left(t + \frac{2}{3}\right)^2 + \frac{13}{3} \end{aligned}

\(t\) の範囲は \(-1 \le t \le 1\) であるから，関数 \(y\) は
\(t = -\dfrac{2}{3}\) のとき 最大値 \(\dfrac{13}{3}\)
\(t = 1\) のとき 最小値 \(-3(1)^2 - 4(1) + 3 = -4\)
をとる。（\(t = -1\) のときは \(y = -3(-1)^2 - 4(-1) + 3 = 4\) である。）
したがって，値域は

\[ -4 \le y \le \frac{13}{3} \]

次に，方程式 \(y-4=0\) すなわち \(y=4\) の解を求める。
\(-3t^2 - 4t + 3 = 4\) より

\begin{aligned} 3t^2 + 4t + 1 &= 0 \\ (3t + 1)(t + 1) &= 0 \\ t &= -1, -\frac{1}{3} \end{aligned}

これらはともに \(-1 \le t \le 1\) を満たす。
\(0 \le x < 2\pi\) において，\(\sin x = t\) を満たす \(x\) の個数を調べる。

  • \(t = -1\) （すなわち \(\sin x = -1\)）のとき，\(x = \dfrac{3}{2}\pi\) の 1個
  • \(t = -\dfrac{1}{3}\) （すなわち \(\sin x = -\dfrac{1}{3}\)）のとき，\(-1 < t < 1\) より \(x\) は 2個存在する。

したがって，求める解の個数は \(1 + 2 = 3\)（個）である。

答: 値域は \(-4 \le y \le \dfrac{13}{3}\)，解の個数は 3個

図解・可視化（定数分離と解の個数）

方程式 \(y = k\)（本問では \(k=4\)）の解の個数は、\(t\) の2次関数のグラフと直線 \(y=k\) の交点の位置によって決まります。
グラフの実線部分が \(t\) のとり得る範囲 (\(-1 \le t \le 1\)) です。
スライダーを動かして直線 \(y=k\) を上下させ、交点の位置に応じて \(x\) の個数がどう変化するか確認してみましょう。

-5 0 5
方程式の解 \(x\) は合計： 0個

最終結果と検算

  • (1) \(y = -4\)
  • (2) \(-1 \le t \le 1\)， \(y = -3t^2 - 4t + 3\)
  • (3) 値域：\(-4 \le y \le \dfrac{13}{3}\)， 解の個数：3個
検算ポイント：(1)の活用と境界条件の確認
(1)で \(x = \dfrac{\pi}{2}\) のとき \(y=-4\) であることを計算しました。このとき \(t = \sin\dfrac{\pi}{2} = 1\) です。
(2)で求めた式に \(t=1\) を代入すると \(y = -3(1)^2 - 4(1) + 3 = -4\) となり、(1)の結果と完全に一致します。これにより、2倍角の公式を用いた式変形が正しかったことが確認できます。
また、\(t=-1\) のとき \(y = -3(1) - 4(-1) + 3 = 4\) となります。方程式 \(y=4\) の解の一つとして \(t=-1\) が出てきたのは、この端点での値と対応しています。

一般化：解の個数問題の全体像

本問の(3)では \(y=4\) の解の個数のみを問われましたが、難関大では「方程式 \(y=k\) の解が4個となる定数 \(k\) の範囲を求めよ」といった形で定数分離法を用いた実数解の配置問題として一般化されて出題されます。
上の「図解・可視化」のスライダーを動かしてわかる通り、\(t\) の2次関数と直線 \(y=k\) の交点のうち、\(-1 < t < 1\) にある交点は「2個」分の働きをし、\(t = \pm 1\) にある交点は「1個」分の働きをします。
例えば、\(k\) が \(-4 < k < 4\) の範囲にあれば、交点は \(-1 < t < 1\) の範囲に2つ存在するため、\(x\) の個数は \(2+2 = 4\)個 となることが直感的に理解できます。このような「文字の置き換えと個数の対応」は、対数関数（\(t=2^x\)など）でも頻出のテーマです。

受験生のつまずきポイント

2倍角の公式の変形ミス

\(\cos 2x = \cos^2 x - \sin^2 x\) をそのまま代入してしまい、\(\cos^2 x\) を \(\sin^2 x\) に直す手間を増やしたり、符号を間違えたりするミスが多発します。式中に \(\sin x\) があるなら、最初から \(\cos 2x = 1 - 2\sin^2 x\) を選択しましょう。

「\(t\)の個数＝\(x\)の個数」と勘違いする

(3)で2次方程式を解いて \(t = -1, -\dfrac{1}{3}\) が出た時点で安心して、「解は2個」と答えてしまう誤答が後を絶ちません。\(t\) はあくまで置き換えた仮の文字であり、最後に必ず「単位円上で \(x\) が何個あるか」を確認するステップを踏んでください。

アクティブリコール（復習テスト）

今日の学習内容を思い出すことで、記憶を定着させましょう。質問をクリックして答えを確認してください。

Q1. 式の中に \(\cos 2x\) と \(\sin x\) が混ざっているとき、最初に行うべき式変形は何ですか？
2倍角の公式を用いて、種類と角を統一します。本問のように \(\sin x\) がある場合は、\(\cos 2x = 1 - 2\sin^2 x\) を用いて式全体を \(\sin\) のみに統一します。
Q2. 変数変換 \(t = \sin x\) を行った直後に、答案に必ず書かなければならないことは何ですか？
新しい変数 \(t\) の「とり得る値の範囲（変域）」です。\(0 \le x < 2\pi\) の場合、単位円を考えて \(-1 \le t \le 1\) と必ず明記します。これがないと最大値や最小値を正しく求められません。
Q3. \(0 \le x < 2\pi\) において、\(\sin x = t\) を満たす \(x\) の個数は、\(t\) の値によってどう変化しますか？
\(t = 1\) または \(t = -1\) のときは \(x\) は1個です。
\(-1 < t < 1\) のときは \(x\) は2個存在します。（範囲外の \(t > 1, t < -1\) のときは0個です。）
