No.11 標準

三角関数

目標解答時間：20分

1 問題

関数 \(y = a\cos^2\theta + 4\sqrt{3}\sin\theta\cos\theta + 3 \cdots\cdots\) ① （\(a\)は定数）がある。また，\(\theta = \dfrac{\pi}{6}\) のとき，\(y=3\) である。

  1. \(a\) の値を求めよ。
  2. ① を \(y = A\sin 2\theta + B\cos 2\theta + C\) （\(A, B, C\) は定数）の形に表せ。
  3. \(\dfrac{\pi}{4} \le \theta \le \dfrac{3}{4}\pi\) における関数①の最大値と最小値，およびそのときの \(\theta\) の値をそれぞれ求めよ。

【条件の整理】

  • 与えられた関数： \(\cos^2\theta\), \(\sin\theta\cos\theta\) という「2次の項」のみが含まれている（定数項は除く）。
  • 通る点： \(\theta = \dfrac{\pi}{6}\) のとき \(y=3\)。これを代入することで未知の定数 \(a\) が確定する。
  • 目標の形： (2)で \(2\theta\) の式に直すことが誘導されている。角を \(\theta\) から \(2\theta\) に「統一」し、さらに(3)で「合成」して1つの三角関数にまとめる流れである。
  • 定義域の制限： \(\theta\) の範囲は \(\dfrac{\pi}{4} \le \theta \le \dfrac{3}{4}\pi\)。最大値・最小値を考える際、合成後の「新しい角」がどの範囲を動くのかを慎重に調べる必要がある。

2 解説

問題を解くための考え方から、答案、検算、発展までを順に確認します。

解法選択のトリガー

2次同次式（\(\sin^2\theta, \cos^2\theta, \sin\theta\cos\theta\) の混合）を見たら

「半角の公式」と「2倍角の公式」を用いて、次数を1次に下げ、角を \(2\theta\) に統一します。
・\(\cos^2\theta = \dfrac{1 + \cos 2\theta}{2}\)
・\(\sin\theta\cos\theta = \dfrac{\sin 2\theta}{2}\)

\(A\sin 2\theta + B\cos 2\theta\) の形になったら

種類が異なる三角関数の和は、そのままでは最大・最小が分かりません。「三角関数の合成」を行って \(R\sin(2\theta + \alpha)\) の形にまとめ、波を1つにします。

最大値・最小値を求めるときは

合成によってできた新しい角（例えば \(2\theta - \dfrac{\pi}{6}\)）を1つの文字とみなし、元の \(\theta\) の範囲から「新しい角の変域」を正確に求めます。単位円を描いて、その範囲内で最も高い位置（最大値）と低い位置（最小値）を探します。

思考プロセス

(1) 代入計算
そのまま \(\theta = \dfrac{\pi}{6}\) を代入します。\(\cos\dfrac{\pi}{6} = \dfrac{\sqrt{3}}{2}\)、\(\sin\dfrac{\pi}{6} = \dfrac{1}{2}\) であることに注意して計算し、\(a\) の1次方程式を解きます。
(2) 次数下げと角の統一
(1)で求めた \(a\) を代入し、\(\cos^2\theta\) を \(\dfrac{1+\cos 2\theta}{2}\) に、\(\sin\theta\cos\theta\) を \(\dfrac{\sin 2\theta}{2}\) に置き換えます。整理して \(A\sin 2\theta + B\cos 2\theta + C\) の形にします。
(3) 合成と変域の確認
得られた式を \(\sin\) に合成します。合成の際、座標平面上の点 \((A, B)\) をプロットして斜辺と角度（偏角）を求めます。
次に、定義域 \(\dfrac{\pi}{4} \le \theta \le \dfrac{3}{4}\pi\) の各辺を「2倍して \(\alpha\) を足す（引く）」ことで、新しい角の範囲を出します。この範囲内での単位円上の \(y\) 座標の最大・最小を読み取ります。

実際の答案

（1）

\(y = a\cos^2\theta + 4\sqrt{3}\sin\theta\cos\theta + 3\) に \(\theta = \dfrac{\pi}{6}, y=3\) を代入すると，

\begin{aligned} 3 &= a\cos^2\frac{\pi}{6} + 4\sqrt{3}\sin\frac{\pi}{6}\cos\frac{\pi}{6} + 3 \\ 3 &= a\left(\frac{\sqrt{3}}{2}\right)^2 + 4\sqrt{3}\left(\frac{1}{2}\right)\left(\frac{\sqrt{3}}{2}\right) + 3 \\ 3 &= \frac{3}{4}a + 4\sqrt{3} \times \frac{\sqrt{3}}{4} + 3 \\ 3 &= \frac{3}{4}a + 3 + 3 \\ \frac{3}{4}a &= -3 \\ a &= -4 \end{aligned}
答: \(a = -4\)

（2）

(1)より，関数①は \(y = -4\cos^2\theta + 4\sqrt{3}\sin\theta\cos\theta + 3\) である。
半角の公式および2倍角の公式より，

\begin{aligned} y &= -4 \left( \frac{1 + \cos 2\theta}{2} \right) + 4\sqrt{3} \left( \frac{\sin 2\theta}{2} \right) + 3 \\ &= -2(1 + \cos 2\theta) + 2\sqrt{3}\sin 2\theta + 3 \\ &= 2\sqrt{3}\sin 2\theta - 2\cos 2\theta + 1 \end{aligned}
答: \(y = 2\sqrt{3}\sin 2\theta - 2\cos 2\theta + 1\)

（3）

(2)で求めた式において，三角関数の合成を行う。
係数の組 \((2\sqrt{3}, -2)\) に対応する距離は \(\sqrt{(2\sqrt{3})^2 + (-2)^2} = \sqrt{12+4} = 4\)
偏角は \(-\dfrac{\pi}{6}\) であるから，

\begin{aligned} y &= 4 \left( \frac{\sqrt{3}}{2}\sin 2\theta - \frac{1}{2}\cos 2\theta \right) + 1 \\ &= 4\sin\left(2\theta - \frac{\pi}{6}\right) + 1 \end{aligned}

ここで，\(\theta\) の変域は \(\dfrac{\pi}{4} \le \theta \le \dfrac{3}{4}\pi\) である。
新しい角 \(2\theta - \dfrac{\pi}{6}\) の変域を求める。

\begin{aligned} \frac{\pi}{4} &\le \theta \le \frac{3}{4}\pi \\ \frac{\pi}{2} &\le 2\theta \le \frac{3}{2}\pi \\ \frac{\pi}{2} - \frac{\pi}{6} &\le 2\theta - \frac{\pi}{6} \le \frac{3}{2}\pi - \frac{\pi}{6} \\ \frac{\pi}{3} &\le 2\theta - \frac{\pi}{6} \le \frac{4}{3}\pi \end{aligned}

\(\alpha = 2\theta - \dfrac{\pi}{6}\) とおくと，\(\dfrac{\pi}{3} \le \alpha \le \dfrac{4}{3}\pi\) の範囲で \(\sin\alpha\) の最大値と最小値を考えればよい。
単位円より，

  • \(\alpha = \dfrac{\pi}{2}\) のとき，\(\sin\alpha\) は最大値 \(1\) をとる。
    このとき \(2\theta - \dfrac{\pi}{6} = \dfrac{\pi}{2} \iff 2\theta = \dfrac{2}{3}\pi \iff \theta = \dfrac{\pi}{3}\)
    \(y\) の最大値は \(4 \times 1 + 1 = 5\)
  • \(\alpha = \dfrac{4}{3}\pi\) のとき，\(\sin\alpha\) は最小値 \(-\dfrac{\sqrt{3}}{2}\) をとる。
    （※範囲内に \(\alpha = \dfrac{3}{2}\pi\) は含まれないため，右端が最小となる）
    このとき \(2\theta - \dfrac{\pi}{6} = \dfrac{4}{3}\pi \iff 2\theta = \dfrac{3}{2}\pi \iff \theta = \dfrac{3}{4}\pi\)
    \(y\) の最小値は \(4 \times \left(-\dfrac{\sqrt{3}}{2}\right) + 1 = 1 - 2\sqrt{3}\)
答:
\(\theta = \dfrac{\pi}{3}\) のとき，最大値 \(5\)
\(\theta = \dfrac{3}{4}\pi\) のとき，最小値 \(1 - 2\sqrt{3}\)

図解・可視化（単位円上の動径の範囲）

三角関数の合成を行った後は、角度を1つのかたまり \(\alpha = 2\theta - \dfrac{\pi}{6}\) とみなし、単位円上で動径がどの範囲を動くかを図示することが極めて重要です。\(\sin\alpha\) は「点\(P\)の\(y\)座標」を表します。

青い太線の部分が、新しい角 \(\alpha = 2\theta - \dfrac{\pi}{6}\) のとり得る範囲です。
この範囲内で、\(\sin\) の値（点\(P\)の高さ）が一番高くなる所と一番低くなる所を探します。

最終結果と検算

  • (1) \(a = -4\)
  • (2) \(y = 2\sqrt{3}\sin 2\theta - 2\cos 2\theta + 1\)
  • (3) \(\theta = \dfrac{\pi}{3}\) で 最大値 \(5\) ， \(\theta = \dfrac{3}{4}\pi\) で 最小値 \(1 - 2\sqrt{3}\)
検算ポイント：端点の代入による確認
定義域の端点である \(\theta = \dfrac{\pi}{4}\) を (2) の式に代入して、最大・最小の範囲内に収まっているか確認します。
\(\theta = \dfrac{\pi}{4}\) のとき：
\(y = 2\sqrt{3}\sin\left(2 \cdot \dfrac{\pi}{4}\right) - 2\cos\left(2 \cdot \dfrac{\pi}{4}\right) + 1 = 2\sqrt{3}\sin\dfrac{\pi}{2} - 2\cos\dfrac{\pi}{2} + 1 = 2\sqrt{3} + 1\)
近似値で考えると、\(\sqrt{3} \approx 1.732\) なので、\(2\sqrt{3} + 1 \approx 4.464\)。
求めた最大値 \(5\) より小さく、最小値 \(1 - 2\sqrt{3} \approx -2.464\) より大きいので、整合性が取れています。

一般化：2次の同次式について

本問のように、式全体が \(\sin^2\theta, \sin\theta\cos\theta, \cos^2\theta\) の項（および定数項）で構成されているものを「\(\sin\theta, \cos\theta\) の2次の同次式（どうじしき）」と呼びます。
関数 \(f(\theta) = a\sin^2\theta + b\sin\theta\cos\theta + c\cos^2\theta\) は、半角の公式と2倍角の公式を用いることで、必ず \(A\sin 2\theta + B\cos 2\theta + C\) の形に変形できます。
したがって、このパターンの式を見たら、ノータイムで「次数下げ $\rightarrow$ 2倍角への統一 $\rightarrow$ 合成」という一連の必勝ルートを思い浮かべられるようにしておきましょう。

受験生のつまずきポイント

半角の公式の符号・係数ミス

(2)の式変形で、\(\cos^2\theta = \dfrac{1 - \cos 2\theta}{2}\) と符号を間違えたり、分母の2を忘れたりするミスが頻発します。\(\cos^2\theta\) は \(\dfrac{1 + \cos 2\theta}{2}\) です（\(\sin^2\theta\) がマイナス）。公式が不安なときは、\(\cos 2\theta = 2\cos^2\theta - 1\) からサッと作れるようにしておきましょう。

合成後の「角度の範囲」の確認を怠る

合成して \(4\sin\left(2\theta - \dfrac{\pi}{6}\right) + 1\) と出た瞬間に安心してしまい、「\(\sin\) の最大は1、最小は-1だから、最大値5、最小値-3」と機械的に答えてしまう誤答が非常に多いです。(3)のように \(\theta\) に定義域がある場合は、必ず「新しい角」の範囲を不等式で作り、単位円で確認する癖をつけてください。

アクティブリコール（復習テスト）

今日の学習内容を思い出すことで、記憶を定着させましょう。質問をクリックして答えを確認してください。

Q1. 式の中に \(\sin^2\theta\), \(\cos^2\theta\), \(\sin\theta\cos\theta\) が混在する「2次の同次式」を見たときの定石となる変形手順は何ですか？
半角の公式と2倍角の公式を用いて次数を1次に下げ、すべての角を \(2\theta\) に統一します。その後、\(A\sin 2\theta + B\cos 2\theta\) の形になるため、三角関数の合成を行って最大・最小を求めます。
Q2. \(A\sin 2\theta + B\cos 2\theta\) を三角関数で合成する際、振幅（係数）\(R\) と偏角 \(\alpha\) はどのように求めますか？
座標平面上に点 \((A, B)\) をプロットします。原点からの距離が振幅 \(R = \sqrt{A^2 + B^2}\) となり、\(x\)軸の正の向きと動径がなす角が偏角 \(\alpha\) となります。これにより \(R\sin(2\theta + \alpha)\) と合成できます。
Q3. 合成して \(y = R\sin(2\theta + \alpha) + C\) の形になった後、最大値・最小値を正しく求めるために必ず行うべき確認作業は何ですか？
\(\theta\) の定義域から、新しい角 \(2\theta + \alpha\) の「変域」を求めることです。求めた範囲を単位円上に図示し、その範囲内で \(y\) 座標（\(\sin\) の値）が最大・最小となる角度を探して計算します。常に \(\pm 1\) をとるとは限らないため注意が必要です。
