No.17 標準

対数関数

目標解答時間：20分

1 問題

関数 $y=(\log_3 x)^2-\log_3 x^2$ がある。また，$u=\log_3 x$ とする。

  1. $y$を$u$を用いて表せ。
  2. $\dfrac{1}{3}\leq x\leq 9$ とする。$u$のとり得る値の範囲を求めよ。またこのとき，$y$の最大値と最小値，およびそのときの$x$の値を求めよ。
  3. $k$を正の定数とする。$x$の方程式 $(\log_3 x)^2-\log_3 x^2=\log_3 k$ が $\dfrac{1}{3}\leq x\leq 9$ の範囲に異なる2つの実数解をもつとき，$k$のとり得る値の範囲を求めよ。またこのとき，異なる2つの実数解を $\alpha, \beta$ ($\alpha<\beta$) とする。$\dfrac{\beta}{\alpha}$ のとり得る値の範囲を求めよ。

【条件の整理】

  • 真数条件： 対数 $\log_3 x$ が含まれるため、真数 $x > 0$ である必要がある。本問では (2)(3) ともに $x \ge \frac{1}{3}$ なので真数条件は満たされている。
  • 対数の性質： $\log_3 x^2$ は、真数条件より $x>0$ が確定しているため、$2\log_3 x$ と変形できる。
  • 変数の置き換え： $u = \log_3 x$ とおくことで、$x$ の複雑な対数方程式を、$u$ のシンプルな2次方程式に帰着させる。
  • $x$ と $u$ の対応関係： 底 $3 > 1$ より、$y = \log_3 x$ は単調増加関数である。したがって、$x$ の値が1つ決まれば $u$ も1つ決まり、$u$ が1つ決まれば $x$ も1つ決まる（1対1対応）。

2 解説

問題を解くための考え方から、答案、検算、発展までを順に確認します。

解法選択のトリガー

同じ形（$\log_3 x$）の繰り返しを見たら

$u = \log_3 x$ と置き換えて2次関数に帰着させます。対数の肩の数字（$x^2$ の $2$）は前におろして形を揃えます。

文字を置き換えたら

「新しい文字の変域」を必ず調べます。元の定義域 $1/3 \le x \le 9$ について、底を3とする対数をとって $u$ の範囲に変換します。

方程式の解の個数と来たら

今回はすでに $y = \log_3 k$ の定数分離の形になっているため、「$u$ の2次関数のグラフと水平線の交点」の個数に翻訳します。$u$ と $x$ が1対1対応することの断り書きを忘れないようにします。

解の比 $\beta/\alpha$ の範囲を求めるときは

置き換えた文字 $u$ の世界で考えます。対数の性質より $\log_3(\beta/\alpha) = \log_3 \beta - \log_3 \alpha$ と「差の形」になるため、グラフ上の2つの交点の $u$ 座標の差（距離）がどう変化するかを調べます。

思考プロセス

(2) 最大値・最小値の計算
$x$ の範囲の各辺に底3の対数をとって $u$ の範囲を出します。$u$ の2次関数を平方完成し、その範囲内での頂点と端点を確認して最大・最小を求めます。
(3) 交点の個数と $k$ の範囲
底が1より大きいため、$x$ についての方程式の実数解の個数は、$u$ についての方程式の実数解の個数と一致します。
$u$ の範囲において、放物線 $y = u^2 - 2u$ と 直線 $y = \log_3 k$ が異なる2点で交わるような $\log_3 k$ の範囲をグラフから読み取り、そこから $k$ の範囲を逆算します。
(3) $\beta/\alpha$ の範囲
$u^2-2u = \log_3 k$ の2つの解を $u_1, u_2$ ($u_1 < u_2$) とします。$\alpha < \beta$ より $\log_3 \alpha < \log_3 \beta$ であるため、$u_1 = \log_3 \alpha, u_2 = \log_3 \beta$ です。これより $\log_3(\beta/\alpha) = u_2 - u_1$ となります。
交点の $u$ 座標の差 $u_2 - u_1$ は、交点が頂点（最小値）から上へ移動するにつれて大きくなります。条件を満たす範囲内で、この差がどこからどこまで動くかを調べ、最後に対数を外して $\beta/\alpha$ の範囲に戻します。

実際の答案

（1）

$y = (\log_3 x)^2 - \log_3 x^2$ において、真数は正であるから $x > 0$ である。
$x>0$ より $\log_3 x^2 = 2\log_3 x$ と変形できる。
$u = \log_3 x$ とおくと，

\[ y = u^2 - 2u \]
答: $y = u^2 - 2u$

（2）

底3は1より大きいので，$\dfrac{1}{3} \le x \le 9$ の各辺に底3の対数をとると

\begin{aligned} \log_3 \frac{1}{3} &\le \log_3 x \le \log_3 9 \\ -1 &\le u \le 2 \end{aligned}

よって，$u$のとり得る値の範囲は $-1 \le u \le 2$ である。
(1)の関数を平方完成すると

\[ y = (u - 1)^2 - 1 \]

したがって，$-1 \le u \le 2$ において関数 $y$ は，
$u = -1$ のとき，最大値 $(-1)^2 - 2(-1) = 1 + 2 = 3$ をとる。
このとき $\log_3 x = -1$ より $x = 3^{-1} = \dfrac{1}{3}$。

$u = 1$ のとき，最小値 $-1$ をとる。
このとき $\log_3 x = 1$ より $x = 3^1 = 3$。

答:
$u$の範囲は $-1 \le u \le 2$
$x = \dfrac{1}{3}$ のとき 最大値 $3$
$x = 3$ のとき 最小値 $-1$

（3）

与えられた方程式は $y = \log_3 k$ と表せる。
底 $3 > 1$ より、$x$ と $u$ は1対1に対応するため、「$x$ の方程式が $\dfrac{1}{3} \le x \le 9$ に異なる2つの実数解をもつ」ことは、「$u$ の方程式 $u^2-2u = \log_3 k$ が $-1 \le u \le 2$ に異なる2つの実数解をもつ」ことと同値である。
これは、放物線 $y = u^2 - 2u$ ($-1 \le u \le 2$) と 直線 $y = \log_3 k$ が異なる2点で交わる条件に等しい。
放物線の頂点の $y$ 座標は $-1$、$u=2$ のときの $y$ 座標は $0$ であるから、グラフより交点を2つもつ条件は

\begin{aligned} -1 &< \log_3 k \le 0 \\ 3^{-1} &< k \le 3^0 \\ \frac{1}{3} &< k \le 1 \end{aligned}

次に、方程式の異なる2つの実数解 $\alpha, \beta$ ($\alpha < \beta$) に対応する $u$ の値を $u_1, u_2$ ($u_1 < u_2$) とすると，$\log_3 \alpha = u_1$，$ \log_3 \beta = u_2$ である。
$\dfrac{\beta}{\alpha}$ を底3の対数をとると

\[ \log_3 \frac{\beta}{\alpha} = \log_3 \beta - \log_3 \alpha = u_2 - u_1 \]

となる。これはグラフ上における2つの交点の $u$ 座標の距離を表す。
放物線 $y = (u-1)^2 - 1$ は軸 $u=1$ に関して対称である。直線 $y = \log_3 k$ が頂点を通るとき（$\log_3 k \to -1$），2つの交点は近づき $u_2 - u_1 \to 0$ となる。
直線が $y=0$ のとき，交点は $u=0, 2$ となり，距離は $u_2 - u_1 = 2 - 0 = 2$ と最大になる。
したがって，交点間の距離のとり得る範囲は

\[ 0 < u_2 - u_1 \le 2 \]

ゆえに

\begin{aligned} 0 &< \log_3 \frac{\beta}{\alpha} \le 2 \\ 3^0 &< \frac{\beta}{\alpha} \le 3^2 \\ 1 &< \frac{\beta}{\alpha} \le 9 \end{aligned}
答: $k$ の範囲は $\dfrac{1}{3} < k \le 1$ ， $\dfrac{\beta}{\alpha}$ の範囲は $1 < \dfrac{\beta}{\alpha} \le 9$

図解・可視化（解の差とグラフの幅）

(3)の後半における $\log_3(\beta/\alpha) = u_2 - u_1$ の処理は、「2次関数のグラフの幅」として視覚的に捉えると非常に簡単です。
水平線 $y = \log_3 k$ を動かすと、交点間の距離 $u_2 - u_1$ が変化します。直線が頂点から上へ行くほど、グラフの幅（解の差）は広がり、最大で $2$ になることがスライダーから直感的に理解できます。

-1 (頂点) 0
解の距離 $u_2 - u_1$ $\left(= \log_3 \dfrac{\beta}{\alpha}\right)$ ： 2.00

最終結果と検算

  • (1) $y = u^2 - 2u$
  • (2) $-1 \le u \le 2$ ， $x = \dfrac{1}{3}$ で最大値 $3$ ， $x = 3$ で最小値 $-1$
  • (3) $k$ の範囲：$\dfrac{1}{3} < k \le 1$ ， $\dfrac{\beta}{\alpha}$ の範囲：$1 < \dfrac{\beta}{\alpha} \le 9$
検算ポイント：解と係数の関係の直接計算
(3)の $\beta/\alpha$ の範囲について、解の公式を用いて直接導くことも可能です。
方程式 $u^2-2u - \log_3 k = 0$ の解は、$u = 1 \pm \sqrt{1 + \log_3 k}$ です。
$-1 < \log_3 k \le 0$ の範囲において、根号の中身 $1+\log_3 k$ は $0 < 1+\log_3 k \le 1$ となります。
したがって、交点の距離 $u_2 - u_1$ は \[ u_2 - u_1 = (1 + \sqrt{1 + \log_3 k}) - (1 - \sqrt{1 + \log_3 k}) = 2\sqrt{1 + \log_3 k} \] 根号の部分が $(0, 1]$ を動くため、$u_2 - u_1$ は $(0, 2]$ を動きます。つまり $0 < \log_3 \dfrac{\beta}{\alpha} \le 2$ となり、対称性から考えた範囲と完全に一致します。

発展・一般化：解の比を扱う定石

対数方程式において、2つの解の「比」$\dfrac{\beta}{\alpha}$ や「積」$\alpha\beta$ を問われる問題は、難関大で頻出します。
・積 $\alpha\beta$ $\rightarrow$ 対数をとると「和」になる（$\log_3(\alpha\beta) = u_1 + u_2$）。解と係数の関係から定数になることが多い。
・比 $\dfrac{\beta}{\alpha}$ $\rightarrow$ 対数をとると「差」になる（$\log_3(\frac{\beta}{\alpha}) = u_2 - u_1$）。解の公式や対称性を利用してグラフの「幅」を評価する。
無理に $x$ に戻して $\alpha, \beta$ を直接求めようとせず、「$u$ の世界で式を立ててから、$x$ の世界に翻訳し直す」という2段階の処理を徹底することが、計算の泥沼化を防ぐ最大のコツです。

受験生のつまずきポイント

真数条件の確認漏れと $\log_3 x^2$ の処理

対数方程式の基本である「真数条件（今回は $x>0, x^2>0$）」を記述し忘れる答案が多いです。一般に $\log_3 x^2 = 2\log_3 |x|$ となりますが、本問は定義域から $x>0$ が確定しているため $|x|$ は不要で $2\log_3 x$ とできます。この理由を論理的に書けるかが部分点の分かれ目になります。

$\dfrac{\beta}{\alpha}$ を直接求めようとして自滅する

$\alpha = 3^{1 - \sqrt{1 + \log_3 k}}$, $\beta = 3^{1 + \sqrt{1 + \log_3 k}}$ と無理やり求めてから割り算をし、指数の複雑な計算でミスをするケースが多発します。「比」を問われたら「対数をとって差にする」という定石を知っているかどうかが、解答時間を大きく左右します。

アクティブリコール（復習テスト）

今日の学習内容を思い出すことで、記憶を定着させましょう。質問をクリックして答えを確認してください。

Q1. $\log_a x^2$ を $2\log_a x$ と変形する際、前提として確認しておかなければならないことは何ですか？
真数条件により「$x > 0$」であることです。一般には $\log_a x^2 = 2\log_a |x|$ となりますが、$x>0$ が保証されている場合のみ絶対値を外して $2\log_a x$ と変形できます。
Q2. $x$ の対数方程式を $u = \log_a x$ と置き換えて解の個数を考える際、答案に必ず明記すべき「$x$ と $u$ の関係」とは何ですか？
「底 $a > 0, a \neq 1$ であるため、$x$ と $u$ は1対1に対応する（実数解の個数が一致する）」という断り書きです。これにより、$u$ の2次方程式の解の個数を調べるだけでよい論理的な根拠となります。
Q3. 対数方程式の2つの実数解を $\alpha, \beta$ とするとき、「解の比 $\dfrac{\beta}{\alpha}$」の範囲をスマートに求めるためにはどのような計算に帰着させますか？
底を同じとする対数をとって、$\log_a\left(\dfrac{\beta}{\alpha}\right) = \log_a \beta - \log_a \alpha$ と「差の形」にします。置き換えた文字 $u$ を用いれば $u_2 - u_1$ となり、2次関数のグラフにおける「2つの交点の距離（幅）」として視覚的かつ容易に範囲を評価できます。
