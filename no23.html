No.23 発展

三角関数

目標解答時間：25分

1 問題

$\theta$についての方程式

\[ 2\sin 2\theta-2k\sin\theta-2k\cos\theta+k^2=0 \quad (k \mbox{ は0以上の定数}) \quad \cdots(*) \]

がある．ただし，$0 \leq \theta < 2\pi$ とする．

  1. $k=0$ のとき，$(*)$ を解け．
  2. $k=1$ のとき，$(*)$ を解け．
  3. $(*)$ が相異なる4個の解をもつような$k$の値の範囲を求めよ．
  4. (3)のとき，$(*)$ の最小の解を $\alpha$ ，最大の解を $\beta$ とする．$\beta$ を $\alpha$ を用いて表せ．

【条件の整理】

  • 方程式の形： $\sin 2\theta$ と $\sin\theta, \cos\theta$ が混在している。まずは倍角の公式 $\sin 2\theta = 2\sin\theta\cos\theta$ で角を $\theta$ に統一する。
  • 定石の落とし穴： $\sin\theta$ と $\cos\theta$ の和を含む対称式を見ると、即座に $t = \sin\theta + \cos\theta$ と置換したくなるが、本問でそれを実行すると $t$ の方程式は解けても $\theta$ に戻す処理が極めて難解になる。
  • 項のグループ分けと因数分解： 展開した式を見ると、項が4つあり、うまく2つずつペアにすると共通因数が見えてくる。これが最大の突破口である。

2 解説

問題を解くための考え方から、答案、検算、発展までを順に確認します。

解法選択のトリガー

項が4つある複雑な方程式

置換を考える前に、まずは「因数分解」を疑います。2項ずつペアにして共通因数でくくる（部分分解）ことで、$(A - B)(C - D) = 0$ の形に持ち込めることが多いです。

解の個数条件

因数分解により $\sin\theta = \frac{k}{2}$ または $\cos\theta = \frac{k}{2}$ となります。これらの解の個数の合計が4個になる条件を考えますが、「共通解（重複）」の存在を忘れないように単位円を描いて確認します。

異なる象限の角の関係式（$\alpha$ と $\beta$）

例えば第4象限の $\beta$ を第1象限の $\alpha$ で表したい場合、$\cos(2\pi - \beta)$ などの公式を用いて、一旦「すべて第1象限の角」に変換して比較するのが定石です。

実際の答案

まず、与えられた方程式 $(*)$ を変形し、因数分解する。
$\sin 2\theta = 2\sin\theta\cos\theta$ より、

\begin{aligned} 4\sin\theta\cos\theta - 2k\sin\theta - 2k\cos\theta + k^2 &= 0 \\ 2\sin\theta(2\cos\theta - k) - k(2\cos\theta - k) &= 0 \\ (2\sin\theta - k)(2\cos\theta - k) &= 0 \end{aligned}

よって、$(*)$ の解は $\sin\theta = \frac{k}{2} \cdots$① または $\cos\theta = \frac{k}{2} \cdots$② を満たす $\theta$ である。

（1）

$k=0$ のとき、①, ② は $\sin\theta = 0$ または $\cos\theta = 0$ となる。
$0 \leq \theta < 2\pi$ の範囲において、
$\sin\theta = 0$ より $\theta = 0, \pi$
$\cos\theta = 0$ より $\theta = \frac{\pi}{2}, \frac{3\pi}{2}$
これらは互いに異なる。

答: $\theta = 0, \frac{\pi}{2}, \pi, \frac{3\pi}{2}$

（2）

$k=1$ のとき、①, ② は $\sin\theta = \frac{1}{2}$ または $\cos\theta = \frac{1}{2}$ となる。
$0 \leq \theta < 2\pi$ の範囲において、
$\sin\theta = \frac{1}{2}$ より $\theta = \frac{\pi}{6}, \frac{5\pi}{6}$
$\cos\theta = \frac{1}{2}$ より $\theta = \frac{\pi}{3}, \frac{5\pi}{3}$

答: $\theta = \frac{\pi}{6}, \frac{\pi}{3}, \frac{5\pi}{6}, \frac{5\pi}{3}$

（3）

$(*)$ が相異なる4個の解をもつためには、方程式①、②がそれぞれ $0 \leq \theta < 2\pi$ の範囲に2個の解をもち、かつそれらに共通解がないことが必要十分条件である。
①、②がそれぞれ2個の解をもつ条件は、$-1 < \frac{k}{2} < 1$ である。
$k \geq 0$ より、$0 \leq \frac{k}{2} < 1$ すなわち $0 \leq k < 2$。

次に、①と②が共通解をもつ条件を考える。
共通解をもつとき $\sin\theta = \cos\theta = \frac{k}{2}$ である。
$\sin^2\theta + \cos^2\theta = 1$ より、

\begin{aligned} \left(\frac{k}{2}\right)^2 + \left(\frac{k}{2}\right)^2 &= 1 \\ \frac{k^2}{2} &= 1 \\ k^2 &= 2 \end{aligned}

$k \geq 0$ より $k = \sqrt{2}$。
このとき $\sin\theta = \cos\theta = \frac{1}{\sqrt{2}}$ となり、$\theta = \frac{\pi}{4}$ が共通解となるため、全体の解の個数は3個に減ってしまう。
したがって、$k = \sqrt{2}$ は除外しなければならない。

答: $0 \leq k < \sqrt{2}$ , $\sqrt{2} < k < 2$

（4）

$\frac{k}{2} = c$ とおく。(3)より $0 \leq c < 1, c \neq \frac{1}{\sqrt{2}}$ である。
①($\sin\theta=c$) の解は第1象限と第2象限に1つずつ、②($\cos\theta=c$) の解は第1象限と第4象限に1つずつ存在する。
$0 \leq \theta < 2\pi$ の範囲において、最小の解 $\alpha$ は第1象限にあり、最大の解 $\beta$ は第4象限にある。すなわち、$\beta$ は必ず $\cos\beta = c$ (②の解) を満たす。

第1象限（$0 \le \theta \le \frac{\pi}{2}$）において、$\alpha$ が①と②のどちらの解に由来するかで場合分けをする。

[i] $0 \leq k < \sqrt{2}$ すなわち $0 \leq c < \frac{1}{\sqrt{2}}$ のとき

第1象限において、$\sin\frac{\pi}{4} = \cos\frac{\pi}{4} = \frac{1}{\sqrt{2}}$ であるから、関数 $\sin\theta$ と $\cos\theta$ のグラフを考えると、解となる角度は小さい方から順に $\sin\theta=c$ の解、$\cos\theta=c$ の解となる。（$\because c < \frac{1}{\sqrt{2}}$）
したがって、最小の解 $\alpha$ は $\sin\alpha = c$ を満たす。
$\beta$ は第4象限の角であり $\cos\beta = c$ を満たすから、

\[ \cos\beta = \cos(2\pi - \beta) = c = \sin\alpha = \cos\left(\frac{\pi}{2} - \alpha\right) \]

ここで、$2\pi - \beta$ と $\frac{\pi}{2} - \alpha$ はともに第1象限の角であるため等しくなる。

\begin{aligned} 2\pi - \beta &= \frac{\pi}{2} - \alpha \\ \beta &= \alpha + \frac{3}{2}\pi \end{aligned}

[ii] $\sqrt{2} < k < 2$ すなわち $\frac{1}{\sqrt{2}} < c < 1$ のとき

同様に第1象限の大小関係を考えると、解となる角度は小さい方から順に $\cos\theta=c$ の解、$\sin\theta=c$ の解となる。
したがって、最小の解 $\alpha$ は $\cos\alpha = c$ を満たす。
$\beta$ は $\cos\beta = c$ を満たす第4象限の角であるから、

\[ \cos\beta = \cos(2\pi - \beta) = c = \cos\alpha \]

$2\pi - \beta$ と $\alpha$ はともに第1象限の角であるため等しくなる。

\begin{aligned} 2\pi - \beta &= \alpha \\ \beta &= -\alpha + 2\pi \end{aligned}
答:
$0 \leq k < \sqrt{2}$ のとき、 $\beta = \alpha + \frac{3}{2}\pi$
$\sqrt{2} < k < 2$ のとき、 $\beta = -\alpha + 2\pi$

図解・可視化（単位円と交点）

(4)の場合分けがなぜ必要なのか、単位円上で視覚的に確認しましょう。
方程式の解は、単位円上の点 $(X, Y) = (\cos\theta, \sin\theta)$ について、
水平線 $Y = k/2$ （$\sin\theta = k/2$に対応） と 垂直線 $X = k/2$ （$\cos\theta = k/2$に対応）
が単位円と交わる点の 偏角（角度） です。
スライダーを動かして $k$ を変化させ、最小の解 $\alpha$（緑色の点）と最大の解 $\beta$（赤色の点）の位置関係の変化を観察してください。

0 $\sqrt{2}\approx 1.41$ 2.0

受験生のつまずきポイント

定石（$t = \sin\theta + \cos\theta$）に固執してタイムアップ

確かに $t$ の方程式に帰着させることはできます（$2t^2 - 2kt + k^2 - 2 = 0$ となる）が、ここから $t$ を求めても、さらに合成して $\theta$ を求めるプロセスが難解で、解の個数や $\alpha, \beta$ の関係を追うのが非常に困難になります。項の形を見たら「まずは因数分解」を試みる柔軟性が求められます。

第4象限の角を第1象限に変換できない

(4)において、$\beta$ が $\cos\beta = c$ を満たすところまではわかっても、$\beta$ と $\alpha$ の関係式を作れない人が多いです。$\cos(2\pi - \beta) = \cos\beta$ を用いて「強制的に第1象限の角 $2\pi - \beta$ に変換して比較する」というテクニックは、難関大で非常によく使われます。

アクティブリコール（復習テスト）

今日の学習内容を思い出すことで、記憶を定着させましょう。質問をクリックして答えを確認してください。

Q1. $\sin 2\theta, \sin\theta, \cos\theta$ が混在する方程式を見たとき、解法の第一手としてまず何を試みますか？
$\sin 2\theta = 2\sin\theta\cos\theta$ と変形し、「因数分解（部分分解）」できないか試みます。それでダメな場合に初めて、$t = \sin\theta + \cos\theta$ などの置換を検討します。
Q2. $0 \le \theta < 2\pi$ において、$\sin\theta = c$ と $\cos\theta = c$ が「共通解」を持つのは、単位円上でどのような点ですか？
$\sin\theta = \cos\theta$ となる点、すなわち単位円と直線 $y=x$ の交点（$\theta = \frac{\pi}{4}, \frac{5\pi}{4}$）です。このとき、解の個数が重なって減ってしまうことに注意が必要です。
Q3. 第1象限の角 $\alpha$ と、第4象限の角 $\beta$ があり、$\cos\beta = \sin\alpha$ を満たしているとき、$\beta$ を $\alpha$ の式で表すとどうなりますか？
第4象限の $\beta$ に対して、第1象限の角 $2\pi - \beta$ を考えます。$\cos(2\pi - \beta) = \cos\beta = \sin\alpha = \cos(\frac{\pi}{2} - \alpha)$ と変形し、ともに第1象限の角として比較すると $2\pi - \beta = \frac{\pi}{2} - \alpha$ となり、$\beta = \alpha + \frac{3\pi}{2}$ が得られます。
