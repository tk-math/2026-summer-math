No.24 標準

平面ベクトル

目標解答時間：25分

1 問題

三角形ABCがあり、辺BCの中点をD、線分ADの中点をEとする。 また、 \[ 0<x<3 \] を満たす実数 $x$ に対して、

\[ x\Vec{AP}+2\Vec{BP}+\Vec{CP}=\vec{0} \]

を満たす点Pを考える。

（1）
（i）
$\Vec{AD}$、$\Vec{AE}$ を $\Vec{AB}$、$\Vec{AC}$ を用いてそれぞれ表せ。
（ii）
$\Vec{AP}$ を $x$、$\Vec{AB}$、$\Vec{AC}$ を用いて表せ。
（iii）
点Pが直線BE上にあるような $x$ の値を求めよ。
（2）
点Qは、 \[ x\Vec{AQ}=3\Vec{BQ} \] を満たす点とする。直線PQは $x$ の値によらず 直線BC上の定点Fを通ることを示し、 $\Vec{AF}$ を $\Vec{AB}$、$\Vec{AC}$ を用いて表せ。

【条件の整理】

  • 始点の統一： 与式には始点が A, B, C と混在しています。ベクトル問題の鉄則として、まずはすべてのベクトルを始点 A に統一（$\Vec{BP} = \Vec{AP} - \Vec{AB}$ など）します。
  • 共線条件（一直線上にある条件）： 「Pが直線BE上にある」という条件は、$\Vec{BP} = s\Vec{BE}$ （$s$ は実数）、あるいは始点をAにして $\Vec{AP} = (1-s)\Vec{AB} + s\Vec{AE}$ と表すことができます。
  • 定点を通る証明： 「$x$ に依らず定点 F を通る」ことを示すには、直線PQ上の点 F を $\Vec{AF} = (1-u)\Vec{AP} + u\Vec{AQ}$ とおいて条件を満たすように $u$ を決めたとき、最終的な $\Vec{AF}$ の式から $x$ が消えて定数だけになることを導きます。

2 解説

問題を解くための考え方から、答案、検算、発展までを順に確認します。

解法選択のトリガー

「$\Vec{AP}$ を表せ」 → 後ろマイナス前の公式で始点をAに統一する
$\Vec{BP}$ や $\Vec{CP}$ を $\Vec{AP} - \Vec{AB}$ , $\Vec{AP} - \Vec{AC}$ と書き換え、$\Vec{AP}$ についての方程式として解きます。
「直線〇〇上にある」 → 2通りのベクトル表現を作り、係数比較する
点Pの位置ベクトルを「(1)(ii)で求めた式」と「直線BE上にある条件から作った式」の2通りで表し、一次独立の断り書きを入れた上で係数を比較します。
「直線BC上の定点F」 → 係数の和が $1$ になる条件を利用する
点 F は直線 PQ 上にあるため $\Vec{AF} = (1-u)\Vec{AP} + u\Vec{AQ}$ と表せます。さらに F が直線 BC 上にあるため、$\Vec{AF} = s\Vec{AB} + t\Vec{AC}$ としたとき $s+t=1$ が成り立ちます。これで $u$ を決定します。

思考プロセス

  • (1) の流れ： (i)は中点の公式。(ii)は始点Aへの書き換えと整理。(iii)は $\Vec{AP} = (1-s)\Vec{AB} + s\Vec{AE}$ とおいて(ii)の式と係数比較を行い、$s$ と $x$ の連立方程式を解きます。
  • (2) の流れ（準備）： まず点 Q の条件 $x\Vec{AQ}=3\Vec{BQ}$ を始点 A に統一し、$\Vec{AQ}$ を $\Vec{AB}$ だけで表します（Q は直線 AB 上の点だとわかります）。
  • (2) の流れ（本筋）： F は直線 PQ 上の点なので実数 $u$ を用いて $\Vec{AF} = (1-u)\Vec{AP} + u\Vec{AQ}$ とおきます。これを展開して $\Vec{AB}$ と $\Vec{AC}$ で整理します。
  • (2) の流れ（決着）： F は直線 BC 上の点なので、「$\Vec{AB}$ の係数 + $\Vec{AC}$ の係数 = $1$」の方程式を立てて $u$ を求めます。求めた $u$ を元の式に代入すると、見事に $x$ が約分されて消え、定数係数だけの $\Vec{AF}$ が得られます。

実際の答案

(1)(i)

点 D は辺 BC の中点であるから、

\[ \Vec{AD} = \frac{\Vec{AB} + \Vec{AC}}{2} \]

点 E は線分 AD の中点であるから、

\[ \Vec{AE} = \frac{1}{2}\Vec{AD} = \frac{1}{2}\left( \frac{\Vec{AB} + \Vec{AC}}{2} \right) = \frac{\Vec{AB} + \Vec{AC}}{4} \]

(1)(ii)

与式 $x\Vec{AP} + 2\Vec{BP} + \Vec{CP} = \vec{0}$ の始点を A に統一すると、

\[ x\Vec{AP} + 2(\Vec{AP} - \Vec{AB}) + (\Vec{AP} - \Vec{AC}) = \vec{0} \] \[ (x + 3)\Vec{AP} = 2\Vec{AB} + \Vec{AC} \]

$0 < x < 3$ より $x + 3 \neq 0$ であるから、

\[ \Vec{AP} = \frac{2\Vec{AB} + \Vec{AC}}{x + 3} \quad \cdots\cdots \text{①} \]

(1)(iii)

点 P は直線 BE 上にあるので、実数 $s$ を用いて $\Vec{BP} = s\Vec{BE}$ と表せる。
始点を A に書き換えると、

\begin{aligned} \Vec{AP} - \Vec{AB} &= s(\Vec{AE} - \Vec{AB}) \\ \Vec{AP} &= (1 - s)\Vec{AB} + s\Vec{AE} \\ &= (1 - s)\Vec{AB} + s\left( \frac{1}{4}\Vec{AB} + \frac{1}{4}\Vec{AC} \right) \\ &= \left( 1 - \frac{3}{4}s \right)\Vec{AB} + \frac{s}{4}\Vec{AC} \quad \cdots\cdots \text{②} \end{aligned}

$\Vec{AB} \neq \vec{0}$, $\Vec{AC} \neq \vec{0}$ であり、$\Vec{AB}$ と $\Vec{AC}$ は平行ではない（一次独立である）から、①と②の係数を比較して、

\[ 1 - \frac{3}{4}s = \frac{2}{x + 3} \quad \cdots\cdots \text{③} \] \[ \frac{s}{4} = \frac{1}{x + 3} \quad \cdots\cdots \text{④} \]

④より $s = \frac{4}{x + 3}$。これを③に代入して、

\[ 1 - \frac{3}{x + 3} = \frac{2}{x + 3} \iff 1 = \frac{5}{x + 3} \]

よって、$x + 3 = 5$ より $x = 2$。
これは $0 < x < 3$ を満たす。

(2)

$x\Vec{AQ} = 3\Vec{BQ}$ の始点を A に統一すると、

\begin{aligned} x\Vec{AQ} &= 3(\Vec{AQ} - \Vec{AB}) \\ (x - 3)\Vec{AQ} &= -3\Vec{AB} \end{aligned}

$0 < x < 3$ より $x - 3 \neq 0$ であるから、

\[ \Vec{AQ} = \frac{-3}{x - 3}\Vec{AB} = \frac{3}{3 - x}\Vec{AB} \]

直線 PQ 上の点 F は、実数 $u$ を用いて $\Vec{AF} = (1 - u)\Vec{AP} + u\Vec{AQ}$ と表せる。

\begin{aligned} \Vec{AF} &= (1 - u)\frac{2\Vec{AB} + \Vec{AC}}{x + 3} + u\frac{3}{3 - x}\Vec{AB} \\ &= \left\{ \frac{2(1 - u)}{x + 3} + \frac{3u}{3 - x} \right\}\Vec{AB} + \frac{1 - u}{x + 3}\Vec{AC} \end{aligned}

点 F が直線 BC 上にあるための条件は、$\Vec{AB}$ と $\Vec{AC}$ の係数の和が $1$ となることであるから、

\[ \frac{2(1 - u)}{x + 3} + \frac{3u}{3 - x} + \frac{1 - u}{x + 3} = 1 \] \[ \frac{3(1 - u)}{x + 3} + \frac{3u}{3 - x} = 1 \]

両辺に $(x + 3)(3 - x)$ を掛けて整理する。

\begin{aligned} 3(1 - u)(3 - x) + 3u(x + 3) &= (x + 3)(3 - x) \\ 3(3 - x - 3u + ux) + 3ux + 9u &= 9 - x^2 \\ 9 - 3x - 9u + 3ux + 3ux + 9u &= 9 - x^2 \\ 9 - 3x + 6ux &= 9 - x^2 \\ 6ux &= 3x - x^2 = x(3 - x) \end{aligned}

$0 < x < 3$ より $x \neq 0$ であるから、両辺を $x$ で割って、

\[ 6u = 3 - x \implies u = \frac{3 - x}{6} \]

このとき、$\Vec{AF}$ の $\Vec{AC}$ の係数は、

\[ \frac{1 - u}{x + 3} = \frac{1 - \frac{3 - x}{6}}{x + 3} = \frac{\frac{3 + x}{6}}{x + 3} = \frac{1}{6} \]

係数の和は $1$ であるから、$\Vec{AB}$ の係数は $1 - \frac{1}{6} = \frac{5}{6}$ となる。
したがって、

\[ \Vec{AF} = \frac{5}{6}\Vec{AB} + \frac{1}{6}\Vec{AC} \]

この式は $x$ を含まず一定である。
ゆえに、直線 PQ は $x$ の値によらず、辺 BC を $1 : 5$ に内分する定点 F を通る。（証明終）

図解・可視化

スライダーで $x$ の値を動かすと、点PはAを通る一定の直線上を、点Qは直線AB上を動きます。
しかし、$x$ がどのように変化しても、直線 PQ は常に辺 BC 上の定点 F を通り続ける ことが視覚的に確認できます。

受験生のつまずきポイント

「一次独立である」ことの記述漏れ

(1)(iii) において、2つのベクトルの式から係数を比較して $s$ と $x$ の連立方程式を作る際、「$\Vec{AB} \neq \vec{0}$, $\Vec{AC} \neq \vec{0}$, $\Vec{AB} \not\parallel \Vec{AC}$ (または一次独立)」という断り書きがないと、厳密には減点対象になります。係数比較を行う際のお決まりの「呪文」として必ず書く癖をつけましょう。

定点を通る証明での $u$ の消去

(2) で「係数和 $= 1$」の方程式を立てて $u$ を求めた後、それを元の式に代入する計算が煩雑になりがちです。ここでは、わざわざ $\Vec{AB}$ の係数に代入して計算しなくても、「係数和が $1$ になるように $u$ を決めたのだから、$\Vec{AC}$ の係数が $\frac{1}{6}$ と分かれば、$\Vec{AB}$ の係数は自動的に $1 - \frac{1}{6} = \frac{5}{6}$ となる」という事実を使うと、計算ミスを防ぎ時間も大幅に短縮できます。

アクティブリコール（本質確認）

答えだけでなく、なぜその式変形や条件が使えるのかを説明してから開いてください。

Q1. 式の中の $\Vec{BP}$ の始点を A にそろえるとき、
どのように変形しますか？
▼
$\Vec{BP} = \Vec{AP} - \Vec{AB}$ と変形します。「後ろの文字マイナス前の文字」の規則で覚えましょう。ベクトル方程式を解く際の最も基本となる第一手です。
Q2. 点 F が直線 BC 上にあるとき、
$\Vec{AF} = s\Vec{AB} + t\Vec{AC}$ の係数 $s, t$ にはどんな関係がありますか？
▼
$s + t = 1$ （係数の和が $1$）になります。この「共線条件」を利用することで、未知数の方程式を立てることができます。
Q3. 「直線 PQ は $x$ の値によらず定点を通る」ことを示すには、
最終的に $\Vec{AF}$ の式がどうなればよいですか？
▼
$\Vec{AF} = \frac{5}{6}\Vec{AB} + \frac{1}{6}\Vec{AC}$ のように、式の係数から変数 $x$ が完全に消えて、「定数」だけで表される状態になれば証明完了です。
