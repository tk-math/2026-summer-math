No.32 標準

高次方程式と軌跡

目標解答時間：25分

1 問題

  1. $a$ ，$b$ は実数の定数とし，$x$ の方程式
    \[ x^3+ax^2+2ax+b=0 \quad \cdots(*) \]
    が $x=1$ を解にもつ．
    1. $b$ を $a$ で表せ．
    2. $a=1$ のとき，$(*)$ を解け．
    3. $(*)$ が虚数解をもつような $a$ の値の範囲を求めよ．
  2. 座標平面上に，定点 $\mathrm{A}(0, -2)$ と2点 $\mathrm{P}$ ，$\mathrm{Q}$ があり，線分 $\mathrm{AP}$ の中点が $\mathrm{Q}$ であるとする．
    1. $\mathrm{P}$ の座標を $(s, t)$ とするとき，$\mathrm{Q}$ の座標を $s$ ，$t$ で表せ．
    2. $\mathrm{Q}$ が放物線 $y=x^2-4x+3$ 上を動くときの $\mathrm{P}$ の軌跡を求め，図示せよ．

【条件の整理】

  • (1) の方程式： 「$x=1$ を解にもつ」という強烈な条件があります。これを代入することで文字を減らし、さらに因数定理で1次式と2次式に分解できることが保証されています。
  • (2) の軌跡： 求めたい軌跡上の点を $\mathrm{P}(s, t)$ とおくのが鉄則です。今回は「$\mathrm{AP}$ の中点が $\mathrm{Q}$」であるため、$\mathrm{Q}$ の座標は $\mathrm{A}$ と $\mathrm{P}$ の中点として $s, t$ の式で表されます。
  • (2) の連動： 点 $\mathrm{Q}$ が特定の図形（放物線）上を動くという条件を、$\mathrm{Q}$ の座標を代入することで処理し、$s$ と $t$ の関係式（これが $\mathrm{P}$ の軌跡）を導き出します。

2 解説

問題を解くための考え方から、答案、検算、発展までを順に確認します。

解法選択のトリガー

3次方程式が虚数解をもつ条件と来たら

3次方程式を因数定理などで (1次式) $\times$ (2次式) = 0 に因数分解します。1次式からは実数解しか出ないため、残りの2次方程式が虚数解をもつ条件（判別式 $D < 0$）に帰着させます。

ある点が動くときの、別の点の軌跡（連動点の軌跡）

① 軌跡を求めたい点 $\mathrm{P}$ を $(x, y)$ [または $(s, t)$] とおく。
② 動く点 $\mathrm{Q}$ を $(X, Y)$ とおき、$\mathrm{P}$ との関係式から $X, Y$ を $x, y$ の式で表す。
③ $X, Y$ を $\mathrm{Q}$ が動く図形の方程式に代入し、$x, y$ の関係式を得る。
これが軌跡問題の黄金の3ステップです。

思考プロセス

(1) の計算手順
(i) $x=1$ を代入して $b$ を $a$ で表す。
(ii) $a=1$ と (i) で求めた $b$ を代入し、3次方程式を因数分解して解く。
(iii) $b$ を消去した3次方程式を組み立てる。$x=1$ を解に持つことがわかっているので、必ず $(x-1)$ で因数分解できる。組立除法などで割り算を行い、現れた2次方程式に $D < 0$ を適用する。
(2) の計算手順
(i) $\mathrm{A}(0, -2)$ と $\mathrm{P}(s, t)$ の中点 $\mathrm{Q}$ の座標を計算する。
(ii) その $\mathrm{Q}$ の座標 $(X, Y)$ を $Y = X^2 - 4X + 3$ に代入する。展開して $t = \dots$ の形に整理すると、$s, t$ が満たすべき方程式、すなわち点 $\mathrm{P}$ の軌跡の方程式が得られる。

実際の答案

（1）(i)

$(*)$ は $x=1$ を解にもつので、$x=1$ を代入すると

\begin{aligned} 1^3 + a\cdot 1^2 + 2a\cdot 1 + b &= 0 \\ 1 + a + 2a + b &= 0 \\ 3a + b + 1 &= 0 \end{aligned}

したがって、$b$ を $a$ で表すと

答: $b = -3a - 1$

（1）(ii)

$a=1$ のとき、(i)より $b = -3(1) - 1 = -4$。
これらを $(*)$ に代入すると

\[ x^3 + x^2 + 2x - 4 = 0 \]

$x=1$ を解にもつことから左辺は $(x-1)$ を因数にもつ。因数分解すると

\[ (x - 1)(x^2 + 2x + 4) = 0 \]

よって $x - 1 = 0$ または $x^2 + 2x + 4 = 0$。
$x^2 + 2x + 4 = 0$ を解の公式で解くと

\[ x = \frac{-2 \pm \sqrt{2^2 - 4\cdot 1\cdot 4}}{2} = \frac{-2 \pm \sqrt{-12}}{2} = \frac{-2 \pm 2\sqrt{3}i}{2} = -1 \pm \sqrt{3}i \]
答: $x = 1, -1 \pm \sqrt{3}i$

（1）(iii)

(i)より $b = -3a - 1$ であるから、$(*)$ は

\[ x^3 + ax^2 + 2ax - 3a - 1 = 0 \]

$x=1$ を解にもつので、左辺を $(x-1)$ で因数分解する。

\[ (x - 1)\{x^2 + (a+1)x + 3a + 1\} = 0 \]

これが虚数解をもつためには、2次方程式 $x^2 + (a+1)x + 3a + 1 = 0$ が虚数解をもてばよい。
その判別式を $D$ とすると、$D < 0$ であるから

\begin{aligned} D &= (a+1)^2 - 4 \cdot 1 \cdot (3a+1) < 0 \\ &= a^2 + 2a + 1 - 12a - 4 < 0 \\ &= a^2 - 10a - 3 < 0 \end{aligned}

2次方程式 $a^2 - 10a - 3 = 0$ の解は、解の公式より

\[ a = \frac{-(-10) \pm \sqrt{(-10)^2 - 4\cdot 1\cdot (-3)}}{2} = \frac{10 \pm \sqrt{112}}{2} = \frac{10 \pm 4\sqrt{7}}{2} = 5 \pm 2\sqrt{7} \]

したがって、不等式の解は

答: $5 - 2\sqrt{7} < a < 5 + 2\sqrt{7}$

（2）(i)

定点 $\mathrm{A}(0, -2)$ と $\mathrm{P}(s, t)$ の中点が $\mathrm{Q}$ であるから、中点の座標公式より、$\mathrm{Q}$ の座標は

\[ \left( \frac{0 + s}{2}, \frac{-2 + t}{2} \right) = \left( \frac{s}{2}, \frac{t - 2}{2} \right) \]
答: $\left( \dfrac{s}{2}, \dfrac{t - 2}{2} \right)$

（2）(ii)

点 $\mathrm{Q}$ は放物線 $y = x^2 - 4x + 3$ 上を動くので、(i)で求めた $\mathrm{Q}$ の座標を代入して

\begin{aligned} \frac{t - 2}{2} &= \left( \frac{s}{2} \right)^2 - 4\left( \frac{s}{2} \right) + 3 \\ \frac{t - 2}{2} &= \frac{s^2}{4} - 2s + 3 \end{aligned}

両辺に 2 を掛けて $t$ について整理すると、

\begin{aligned} t - 2 &= \frac{s^2}{2} - 4s + 6 \\ t &= \frac{1}{2}s^2 - 4s + 8 \end{aligned}

したがって、点 $\mathrm{P}(s, t)$ は放物線 $y = \dfrac{1}{2}x^2 - 4x + 8$ 上にある。
逆に、この放物線上の任意の点は条件を満たす。

答: 軌跡は放物線 $y = \dfrac{1}{2}x^2 - 4x + 8$

【図示】

軌跡の放物線を平方完成すると $y = \dfrac{1}{2}(x^2 - 8x) + 8 = \dfrac{1}{2}(x - 4)^2$ となる。
頂点は $(4, 0)$、$y$ 切片は $8$ である。これを図示する。（図は下図の通り）

図解・可視化（連動点の軌跡）

「$\mathrm{AP}$ の中点が $\mathrm{Q}$ である」ということは、逆に言えば「点 $\mathrm{P}$ は、定点 $\mathrm{A}$ を基準として点 $\mathrm{Q}$ を2倍に拡大した位置にある」ということです。
スライダーを動かすと、点 $\mathrm{Q}$ が元の放物線 $C_1$ 上を動き、それに連動して点 $\mathrm{P}$ が新しい放物線 $C_2$（軌跡）を描き出す様子が観察できます。

最終結果と検算

  • (1)(i) $b = -3a - 1$
  • (1)(ii) $x = 1, -1 \pm \sqrt{3}i$
  • (1)(iii) $5 - 2\sqrt{7} < a < 5 + 2\sqrt{7}$
  • (2)(i) $\mathrm{Q}\left( \dfrac{s}{2}, \dfrac{t - 2}{2} \right)$
  • (2)(ii) 軌跡は放物線 $y = \dfrac{1}{2}x^2 - 4x + 8$
検算ポイント：特定の点を代入して確認する
(2)で求めた軌跡の式が正しいか、具体的な点を用いて確認します。
放物線 $\mathrm{Q}$ の頂点は $y = (x-2)^2 - 1$ より $(2, -1)$ です。
点 $\mathrm{Q}$ が $(2, -1)$ にいるとき、定点 $\mathrm{A}(0, -2)$ と点 $\mathrm{P}(s, t)$ の中点が $(2, -1)$ になるため、
$\dfrac{0 + s}{2} = 2 \implies s = 4$
$\dfrac{-2 + t}{2} = -1 \implies -2 + t = -2 \implies t = 0$
よって、点 $\mathrm{P}$ は $(4, 0)$ を通るはずです。
求めた軌跡の方程式 $y = \dfrac{1}{2}x^2 - 4x + 8$ に $x=4$ を代入すると、
$y = \dfrac{1}{2}(16) - 16 + 8 = 8 - 16 + 8 = 0$ となり、確かに $(4, 0)$ を通ります。式変形が間違っていないことが証明できました。

受験生のつまずきポイント

(1) 因数定理による割り算の実行

$x=1$ を解に持つことが分かっているのに、$x^3+ax^2+2ax-3a-1$ を因数分解する際に手が止まってしまう生徒がいます。係数に文字 $a$ が含まれていても、筆算や組立除法で必ず $(x-1)$ で割り切れます（余りが0になります）。信じて計算を進めましょう。

(2) $s, t$ の代入方向を逆にしてしまう

$\mathrm{Q}\left(\dfrac{s}{2}, \dfrac{t-2}{2}\right)$ と求めた後、これを「$x = \dfrac{s}{2}, y = \dfrac{t-2}{2}$」として放物線の式に代入するのが正解ですが、逆に「$s = \dfrac{x}{2}$」のように元の放物線の式に $x, y$ をそのまま残して代入してしまうミスが頻発します。「動く点（今回は $\mathrm{Q}$）の座標を、その点が動く図形の式に代入する」という原則を徹底してください。

Q1. 3次方程式が「虚数解をもつ」条件を求める際、まず行うべき変形は何ですか？
因数定理などを用いて、方程式を「$(1次式) \times (2次式) = 0$」の形に因数分解します。1次式から出る解は必ず実数になるため、残った「2次方程式の判別式 $D < 0$」を条件として処理します。
Q2. 軌跡の問題において、「点 $\mathrm{Q}$ がある図形上を動くとき、点 $\mathrm{P}$ の軌跡を求めよ」という連動点の軌跡を求める手順（3ステップ）を答えてください。
① 求めたい点 $\mathrm{P}$ を $(s, t)$ とおき、動く点 $\mathrm{Q}$ の座標を $s, t$ を用いて表す。
② $\mathrm{Q}$ の座標を、$\mathrm{Q}$ が動く図形の方程式（$x, y$）に代入する。
③ $s, t$ の関係式として整理し、最後に $s \to x, t \to y$ に書き換えて図形の方程式とする。
Q3. 点 $\mathrm{A}(x_A, y_A)$ と 点 $\mathrm{P}(s, t)$ の中点が $\mathrm{Q}$ であるとき、$\mathrm{Q}$ の座標はどのように表されますか？
中点の座標公式より、それぞれの $x$ 座標の平均、$y$ 座標の平均をとって $\left( \dfrac{x_A + s}{2}, \dfrac{y_A + t}{2} \right)$ と表されます。
