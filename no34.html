No.34 標準

対数関数

目標解答時間：20分

1 問題

$x$ の方程式

\[ (\log_2 x)^2+\log_2 x^2+a=0 \quad \cdots(*) \]

がある。ただし，$a$ は実数の定数である。

  1. $s=\log_2 x$ とおくとき，$(*)$ を $s$ を用いて表せ。
  2. $(*)$ が異なる2つの実数解をもつような $a$ の値の範囲を求めよ。
  3. (2)のときの $(*)$ の解を $x=\alpha$ ，$\beta \ (\alpha<\beta)$ とする。
    $t$ の方程式 \[ 2^{2t+2}-5\cdot2^t+b=0 \quad (b \mbox{ は実数の定数}) \] の解が $t=\log_2 \alpha$ ，$\log_2 \beta$ となるような $a$ ，$b$ の値を求めよ。

【条件の整理】

  • 真数条件の確認： $\log_2 x$ と $\log_2 x^2$ があるため、真数は正（$x>0$ かつ $x^2>0$）。まとめると $x>0$ が大前提となります。
  • 変数の置き換え： 前半は $s = \log_2 x$ の2次方程式、後半は $X = 2^t$ の2次方程式に帰着させます。
  • 解の対応関係： $\alpha, \beta$ は $x$ の解ですが、$s$ の方程式の解は $s_1 = \log_2 \alpha, s_2 = \log_2 \beta$ となります。これがそのまま後半の $t$ の解になるという構造を読み取ります。

2 解説

問題を解くための考え方から、答案、検算、発展までを順に確認します。

解法選択のトリガー

$\log_a x^2$ の処理と来たら

真数条件 $x>0$ を確認した上で、肩の数字を前に下ろして $2\log_a x$ と変形し、同じパーツ（$\log_a x$）を揃えます。

方程式の解の個数と来たら

$s = \log_2 x$ と置いたとき、底 $2 > 1$ より $x>0$ において $x$ と $s$ は1対1に対応します。したがって、$x$ の解の個数はそのまま $s$ の2次方程式の解の個数（判別式 $D$）と一致します。

2つの解の「積」や「和」と来たら

「解と係数の関係」を利用して橋渡しをします。本問は、前半の $s$ の方程式での解と係数の関係と、後半の $X$ の方程式での解と係数の関係を連立させることで、一気に $a$ と $b$ が求まる美しい構造になっています。

思考プロセス

(1)(2) の計算手順
$x>0$ より $\log_2 x^2 = 2\log_2 x$ となり、$(*)$ は $(\log_2 x)^2 + 2\log_2 x + a = 0$ になります。$s = \log_2 x$ とおけば $s^2 + 2s + a = 0$ です。この2次方程式が異なる2つの実数解をもつ条件として、判別式 $D/4 = 1 - a > 0$ を解きます。
(3) 前半と後半のリンク
前半の $s$ の方程式の解を $s_1 = \log_2 \alpha$, $s_2 = \log_2 \beta$ とおくと、解と係数の関係から $s_1 + s_2 = -2$ と $s_1 s_2 = a$ が得られます。
後半の方程式 $2^{2t+2} - 5\cdot 2^t + b = 0$ を $X = 2^t$ とおいて $4X^2 - 5X + b = 0$ とします。
この解は $X_1 = 2^{s_1} = 2^{\log_2 \alpha} = \alpha$、同様に $X_2 = \beta$ となります。
(3) $a, b$ を決定する仕上げ
後半の $X$ の方程式における解と係数の関係から、$\alpha + \beta = 5/4$、$\alpha\beta = b/4$ が成り立ちます。
前半の $s_1 + s_2 = \log_2\alpha + \log_2\beta = \log_2(\alpha\beta) = -2$ を利用すると、$\alpha\beta = 2^{-2} = 1/4$ が確定します。これにより $b = 1$ が出ます。
$b=1$ を元の式 $4X^2 - 5X + 1 = 0$ に戻して解くと $\alpha, \beta$ の値が直接求まり、そこから $s_1, s_2$、そして $a = s_1 s_2$ が芋づる式に求まります。

実際の答案

（1）

対数の真数条件より、$x > 0$ かつ $x^2 > 0$ であるから、$x > 0$。
これより、$\log_2 x^2 = 2\log_2 x$ と変形できる。よって $(*)$ は

\[ (\log_2 x)^2 + 2\log_2 x + a = 0 \]

$s = \log_2 x$ とおくと、

答: $s^2 + 2s + a = 0$

（2）

$x > 0$ において、$s = \log_2 x$ はすべての実数値をとる。
底 $2 > 1$ より $x$ と $s$ は1対1に対応するため、$(*)$ が異なる2つの実数解をもつ条件は、(1)で求めた $s$ の2次方程式が異なる2つの実数解をもつことと同値である。
$s^2 + 2s + a = 0$ の判別式を $D$ とすると、$D > 0$ であればよいから、

\begin{aligned} \frac{D}{4} = 1^2 - 1 \cdot a &> 0 \\ 1 - a &> 0 \\ a &< 1 \end{aligned}
答: $a < 1$

（3）

方程式 $s^2 + 2s + a = 0$ の2つの解を $s_1, s_2 \ (s_1 < s_2)$ とする。
$\alpha < \beta$ であり、底 $2 > 1$ より単調増加であるから、$s_1 = \log_2 \alpha, \ s_2 = \log_2 \beta$ である。
解と係数の関係より、

\begin{aligned} s_1 + s_2 &= -2 \quad \cdots \text{①} \\ s_1 s_2 &= a \quad \cdots \text{②} \end{aligned}

一方、$t$ の方程式 $2^{2t+2} - 5\cdot 2^t + b = 0$ を変形する。

\[ 4(2^t)^2 - 5\cdot 2^t + b = 0 \]

$X = 2^t$ とおくと、$4X^2 - 5X + b = 0$ となる。
この方程式の解は、条件より $t = \log_2 \alpha = s_1$ および $t = \log_2 \beta = s_2$ を代入したものであるから、

\begin{aligned} X &= 2^{s_1} = 2^{\log_2 \alpha} = \alpha \\ X &= 2^{s_2} = 2^{\log_2 \beta} = \beta \end{aligned}

すなわち、2次方程式 $4X^2 - 5X + b = 0$ の解は $\alpha$ と $\beta$ である。
解と係数の関係より、

\begin{aligned} \alpha + \beta &= \frac{5}{4} \quad \cdots \text{③} \\ \alpha \beta &= \frac{b}{4} \quad \cdots \text{④} \end{aligned}

ここで、式①より

\begin{aligned} \log_2 \alpha + \log_2 \beta &= -2 \\ \log_2 (\alpha \beta) &= -2 \\ \alpha \beta &= 2^{-2} = \frac{1}{4} \end{aligned}

これを式④に代入すると、$\dfrac{b}{4} = \dfrac{1}{4} \implies \boldsymbol{b = 1}$。

$b=1$ を $X$ の2次方程式に戻すと、

\begin{aligned} 4X^2 - 5X + 1 &= 0 \\ (4X - 1)(X - 1) &= 0 \\ X &= \frac{1}{4}, \ 1 \end{aligned}

$\alpha < \beta$ であるから、$\alpha = \dfrac{1}{4}$, $\beta = 1$ と確定する。（和が $5/4$ になるため式③も満たす）
これにより、$s_1, s_2$ の値は

\begin{aligned} s_1 &= \log_2 \frac{1}{4} = -2 \\ s_2 &= \log_2 1 = 0 \end{aligned}

式②より、$a = s_1 s_2 = (-2) \cdot 0 = \boldsymbol{0}$。
これは (2)の条件 $a < 1$ を満たす。

答: $a = 0, \ b = 1$

図解・可視化（パラメータ $a$ と方程式の連動）

前半のパラメータ $a$ を動かすと、$s$ の方程式の解 $s_1, s_2$ が変化し、それに伴って $\alpha, \beta$ の値も連動して動きます。
グラフ上の青い実線は後半の目標となる方程式 $y = 4X^2 - 5X + 1$ を表し、赤い破線は $a$ の変化から作られる方程式 $y = 4X^2 - 4(\alpha+\beta)X + 4\alpha\beta$ を表します。

別解の考察

対称式と相加相乗平均を用いた $\alpha, \beta$ の絞り込み
答案では $b=1$ を求めてから $4X^2 - 5X + 1 = 0$ を解きましたが、$b$ を経由せずに $\alpha, \beta$ の関係式から直接解を求めることも可能です。
$s_1 = -1 - \sqrt{1-a}$, \ $s_2 = -1 + \sqrt{1-a}$ より、
$\alpha = 2^{-1-\sqrt{1-a}}$, \ $\beta = 2^{-1+\sqrt{1-a}}$ と表せます。
このとき、和は $\alpha + \beta = 2^{-1}(2^{-\sqrt{1-a}} + 2^{\sqrt{1-a}}) = \dfrac{1}{2}(2^{\sqrt{1-a}} + 2^{-\sqrt{1-a}})$ となります。
これが条件より $\dfrac{5}{4}$ に等しいので、
$2^{\sqrt{1-a}} + 2^{-\sqrt{1-a}} = \dfrac{5}{2}$
$Y = 2^{\sqrt{1-a}} \ (Y \ge 1)$ とおくと、$Y + \dfrac{1}{Y} = \dfrac{5}{2}$ となり、両辺に $2Y$ を掛けて整理すると
$2Y^2 - 5Y + 2 = 0 \implies (2Y-1)(Y-2) = 0$
$Y \ge 1$ より $Y = 2$。
すなわち $2^{\sqrt{1-a}} = 2^1 \implies \sqrt{1-a} = 1 \implies 1-a = 1 \implies a=0$ と求まります。
このアプローチは計算量はやや増えますが、指数関数の対称な構造（$Y + 1/Y$ の形）を深く理解するのに非常に役立ちます。

受験生のつまずきポイント

$\log_2 x^2$ を $2\log_2 x$ にする前の「真数条件の確認」漏れ

一般に $\log_a x^2 = 2\log_a |x|$ であり、絶対値がつきます。本問は初めから $(\log_2 x)^2$ が存在するため、暗黙のうちに真数条件 $x>0$ が確定しており、絶対値を外して $2\log_2 x$ とできます。このロジックを答案に書かずにいきなり変形すると、厳密な採点では減点される恐れがあります。

$\alpha\beta$ と $s_1 s_2$ の混同

後半で $s_1 s_2 = a$ と $\alpha\beta = b/4$ が出てきますが、この2つを直接イコールで結んでしまうミスが頻発します。$\alpha$ と $s_1$ は対数の関係（$s_1 = \log_2 \alpha$）で結ばれているため、和と積が変換されます（対数の和は真数の積）。ここを丁寧に整理できるかが合否を分けます。

アクティブリコール（復習テスト）

Q1. $\log_a x^2$ を $2\log_a x$ と変形する際、前提として確認しておかなければならないことは何ですか？
真数条件により「$x > 0$」であることです。一般には $\log_a x^2 = 2\log_a |x|$ となりますが、$x>0$ が保証されている場合のみ絶対値を外して $2\log_a x$ と変形できます。
Q2. $x$ の対数方程式を $s = \log_a x$ と置き換えて解の個数を考える際、答案に必ず明記すべき「$x$ と $s$ の関係」とは何ですか？
「底 $a > 0, a \neq 1$ であるため、$x$ と $s$ は1対1に対応する（実数解の個数が一致する）」という断り書きです。これにより、$s$ の2次方程式の解の個数を調べるだけでよい論理的な根拠となります。
Q3. 2つの解の積 $\alpha\beta$ の値を知りたいとき、対数を用いた式 $\log_a \alpha$ と $\log_a \beta$ について、どのような計算をすればよいですか？
2つの式を「足し算」します。対数の性質より $\log_a \alpha + \log_a \beta = \log_a (\alpha\beta)$ となるため、和の値から真数の積を逆算することができます。
