No.35 標準

空間ベクトル

目標解答時間：20分

1 問題

体積が1である四面体 $\mathrm{OABC}$ があり，三角形 $\mathrm{OBC}$ の重心を $\mathrm{G}$ とし，線分 $\mathrm{AG}$ の中点を $\mathrm{M}$ とする。

  1. $\Vec{OG}$ ，$\Vec{OM}$ を $\Vec{OA}$ ，$\Vec{OB}$ ，$\Vec{OC}$ で表せ。
  2. 直線 $\mathrm{BM}$ と平面 $\mathrm{OAC}$ の交点を $\mathrm{N}$ とする。
    1. $\Vec{ON}$ を $\Vec{OA}$ ，$\Vec{OC}$ で表せ。
    2. 四面体 $\mathrm{OAMN}$ の体積を求めよ。

【条件の整理】

  • 基準となるベクトル： 空間内の任意の点は、1次独立な3つのベクトル $\Vec{OA}, \Vec{OB}, \Vec{OC}$ を用いて一意に表すことができます。これを解答の基本骨格とします。
  • 直線の条件： 点 $\mathrm{N}$ が直線 $\mathrm{BM}$ 上にある $\Rightarrow$ $\Vec{ON} = (1-s)\Vec{OB} + s\Vec{OM}$ と実数 $s$ で表せる。
  • 平面の条件： 点 $\mathrm{N}$ が平面 $\mathrm{OAC}$ 上にある $\Rightarrow$ $\Vec{ON}$ を $\Vec{OA}, \Vec{OB}, \Vec{OC}$ で表したとき、$\Vec{OB}$ の係数が $0$ になる。
  • 体積比： 四面体 $\mathrm{OAMN}$ と 四面体 $\mathrm{OABC}$ の体積比を考える際、「底面をどこにとるか」が重要です。両者が共有している平面（今回は平面 $\mathrm{OAC}$）上の三角形を底面とすると、見通しが良くなります。

2 解説

問題を解くための考え方から、答案、検算、発展までを順に確認します。

解法選択のトリガー

「直線と平面の交点」と来たら

① 直線上にある条件（係数の和が1、または媒介変数表示）、② 平面上にある条件（不要なベクトルの係数が0、または係数和が1）の2つを組み合わせて連立方程式を立てます。空間ベクトルの最重要テーマです。

四面体の「体積比」を求めるときは

体積比 ＝ 「底面積の比」 $\times$ 「高さの比」 に分解して考えます。
どの面を底面に選ぶかで難易度が激変します。基本は「共通の頂点（または平面）を含む面」を底面に設定します。本問では平面 $\mathrm{OAC}$ に乗っている $\triangle\mathrm{OAC}$ と $\triangle\mathrm{OAN}$ を底面とします。

思考プロセス

(1) の計算手順
重心の公式 $\Vec{OG} = \dfrac{\Vec{OO} + \Vec{OB} + \Vec{OC}}{3} = \dfrac{\Vec{OB} + \Vec{OC}}{3}$ を用います。
$\mathrm{M}$ は $\mathrm{A}$ と $\mathrm{G}$ の中点なので、中点の公式 $\Vec{OM} = \dfrac{\Vec{OA} + \Vec{OG}}{2}$ に代入するだけです。
(2)(i) の計算手順
$\mathrm{N}$ は直線 $\mathrm{BM}$ 上より $\Vec{ON} = (1-s)\Vec{OB} + s\Vec{OM}$。
ここに(1)の $\Vec{OM}$ を代入し、$\Vec{OA}, \Vec{OB}, \Vec{OC}$ の形に整理します。
$\mathrm{N}$ は平面 $\mathrm{OAC}$ 上にあるため、式の中から $\Vec{OB}$ が消えなければなりません。つまり $\Vec{OB}$ の係数 $= 0$ として $s$ を求めます。
(2)(ii) 体積比の分解
・底面積の比（$\triangle\mathrm{OAN} : \triangle\mathrm{OAC}$）： (2)(i) で求めた $\Vec{ON}$ の式から、$\mathrm{N}$ が $\triangle\mathrm{OAC}$ 内でどのような位置にあるかを調べます。内分点の公式を逆算して線分比を出し、面積比を求めます。
・高さの比（頂点 $\mathrm{M}$ と 頂点 $\mathrm{B}$）： 点 $\mathrm{M}$ が線分 $\mathrm{BN}$ 上のどこにいるかで決まります。$s$ の値（$\Vec{ON}$ の式）から、$\mathrm{M}$ が $\mathrm{BN}$ を何対何に内分しているか読み取り、平面 $\mathrm{OAC}$ までの距離（高さ）の比を求めます。
最後に 面積比 $\times$ 高さの比 を計算して終了です。

実際の答案

（1）

点 $\mathrm{G}$ は $\triangle\mathrm{OBC}$ の重心であるから、

\[ \Vec{OG} = \frac{\Vec{OB} + \Vec{OC}}{3} \]

点 $\mathrm{M}$ は線分 $\mathrm{AG}$ の中点であるから、

\begin{aligned} \Vec{OM} &= \frac{\Vec{OA} + \Vec{OG}}{2} \\ &= \frac{1}{2}\Vec{OA} + \frac{1}{2}\left(\frac{\Vec{OB} + \Vec{OC}}{3}\right) \\ &= \frac{1}{2}\Vec{OA} + \frac{1}{6}\Vec{OB} + \frac{1}{6}\Vec{OC} \end{aligned}
答: $\Vec{OG} = \dfrac{1}{3}\Vec{OB} + \dfrac{1}{3}\Vec{OC} , \quad \Vec{OM} = \dfrac{1}{2}\Vec{OA} + \dfrac{1}{6}\Vec{OB} + \dfrac{1}{6}\Vec{OC}$

（2）(i)

点 $\mathrm{N}$ は直線 $\mathrm{BM}$ 上にあるので、実数 $s$ を用いて次のように表せる。

\begin{aligned} \Vec{ON} &= (1-s)\Vec{OB} + s\Vec{OM} \\ &= (1-s)\Vec{OB} + s\left(\frac{1}{2}\Vec{OA} + \frac{1}{6}\Vec{OB} + \frac{1}{6}\Vec{OC}\right) \\ &= \frac{s}{2}\Vec{OA} + \left(1 - s + \frac{s}{6}\right)\Vec{OB} + \frac{s}{6}\Vec{OC} \\ &= \frac{s}{2}\Vec{OA} + \left(1 - \frac{5}{6}s\right)\Vec{OB} + \frac{s}{6}\Vec{OC} \end{aligned}

四面体 $\mathrm{OABC}$ において、$\Vec{OA}, \Vec{OB}, \Vec{OC}$ は1次独立であり、点 $\mathrm{N}$ は平面 $\mathrm{OAC}$ 上にあるから、$\Vec{OB}$ の係数は $0$ である。

\begin{aligned} 1 - \frac{5}{6}s &= 0 \\ \frac{5}{6}s &= 1 \implies s = \frac{6}{5} \end{aligned}

これを $\Vec{ON}$ の式に代入して、

\begin{aligned} \Vec{ON} &= \frac{1}{2}\left(\frac{6}{5}\right)\Vec{OA} + \frac{1}{6}\left(\frac{6}{5}\right)\Vec{OC} \\ &= \frac{3}{5}\Vec{OA} + \frac{1}{5}\Vec{OC} \end{aligned}
答: $\Vec{ON} = \dfrac{3}{5}\Vec{OA} + \dfrac{1}{5}\Vec{OC}$

（2）(ii)

四面体 $\mathrm{OAMN}$ と 四面体 $\mathrm{OABC}$ について、平面 $\mathrm{OAC}$ に含まれる面を底面として体積比を考える。

[底面積の比]
(i)より $\Vec{ON} = \dfrac{3}{5}\Vec{OA} + \dfrac{1}{5}\Vec{OC} = \dfrac{4}{5} \left( \dfrac{3\Vec{OA} + \Vec{OC}}{4} \right)$。
線分 $\mathrm{AC}$ を $1:3$ に内分する点を $\mathrm{P}$ とすると、$\Vec{OP} = \dfrac{3\Vec{OA} + \Vec{OC}}{4}$ である。
したがって、$\Vec{ON} = \dfrac{4}{5}\Vec{OP}$ となり、点 $\mathrm{N}$ は線分 $\mathrm{OP}$ を $4:1$ に内分する点である。
これより、面積比は以下のようになる。

\begin{aligned} \triangle\mathrm{OAN} &= \frac{4}{5} \triangle\mathrm{OAP} \\ &= \frac{4}{5} \left( \frac{1}{4} \triangle\mathrm{OAC} \right) = \frac{1}{5} \triangle\mathrm{OAC} \end{aligned}

[高さの比]
頂点 $\mathrm{M}$ および 頂点 $\mathrm{B}$ から平面 $\mathrm{OAC}$ に下ろした垂線の長さの比は、線分 $\mathrm{BN}$ 上における $\mathrm{M}$ の位置で決まる。
(i)において $\Vec{ON} = (1-s)\Vec{OB} + s\Vec{OM}$ とおき、$s = \dfrac{6}{5}$ を得た。
これより $\Vec{ON} = -\dfrac{1}{5}\Vec{OB} + \dfrac{6}{5}\Vec{OM}$ となるから、変形すると

\begin{aligned} 5\Vec{ON} &= -\Vec{OB} + 6\Vec{OM} \\ \Vec{OM} &= \frac{\Vec{OB} + 5\Vec{ON}}{6} \end{aligned}

これは、点 $\mathrm{M}$ が線分 $\mathrm{BN}$ を $5:1$ に内分することを示している。
したがって、点 $\mathrm{N}$（平面 $\mathrm{OAC}$ 上の点）から $\mathrm{M}$ までの距離は、$\mathrm{N}$ から $\mathrm{B}$ までの距離の $\dfrac{1}{6}$ 倍であるから、高さも $\dfrac{1}{6}$ 倍となる。

[体積の計算]
四面体 $\mathrm{OAMN}$ の体積 $V'$ は、四面体 $\mathrm{OABC}$ の体積 $V$ に対して

\begin{aligned} V' &= V \times (\text{底面積の比}) \times (\text{高さの比}) \\ &= 1 \times \frac{1}{5} \times \frac{1}{6} \\ &= \frac{1}{30} \end{aligned}
答: $\dfrac{1}{30}$

図解・可視化（体積比の分解）

四面体の体積比は、「底面（平面$\mathrm{OAC}$）」における縮小率と、「高さ（直線$\mathrm{BN}$）」における縮小率に分けて視覚化すると分かりやすいです。
ボタンを切り替えて、それぞれの視点から比率を確認してみましょう。

平面 OAC を真上から見た図（底面積の比）

$\triangle\mathrm{OAN}$ は $\triangle\mathrm{OAP}$ の $\dfrac{4}{5}$倍。
$\triangle\mathrm{OAP}$ は $\triangle\mathrm{OAC}$ の $\dfrac{1}{4}$倍。
ゆえに底面積は $\dfrac{4}{5} \times \dfrac{1}{4} = \mathbf{\dfrac{1}{5}}$ 倍。

線分 BN を横から見た図（高さの比）

点 $\mathrm{M}$ は線分 $\mathrm{BN}$ を $5:1$ に内分する。
平面 $\mathrm{OAC}$（点 $\mathrm{N}$ を含む面）からの高さは、
相似比により $\mathrm{B}$ の高さの $\mathbf{\dfrac{1}{6}}$ 倍となる。

最終結果と検算

  • (1) $\Vec{OG} = \dfrac{1}{3}\Vec{OB} + \dfrac{1}{3}\Vec{OC}$ ， $\Vec{OM} = \dfrac{1}{2}\Vec{OA} + \dfrac{1}{6}\Vec{OB} + \dfrac{1}{6}\Vec{OC}$
  • (2)(i) $\Vec{ON} = \dfrac{3}{5}\Vec{OA} + \dfrac{1}{5}\Vec{OC}$
  • (2)(ii) 体積 $\dfrac{1}{30}$
別解：公式を用いた一発計算（ベクトルと体積比）
四面体 $\mathrm{OABC}$ と 四面体 $\mathrm{OPQR}$ があり、頂点 $\mathrm{O}$ を共有しているとき、それぞれの側稜上（またはその延長上）にある点 $\mathrm{P, Q, R}$ について、体積比は以下の公式で求まります。
\[ \frac{V_{\mathrm{OPQR}}}{V_{\mathrm{OABC}}} = \frac{\mathrm{OP}}{\mathrm{OA}} \times \frac{\mathrm{OQ}}{\mathrm{OB}} \times \frac{\mathrm{OR}}{\mathrm{OC}} \] 本問の四面体 $\mathrm{OAMN}$ にこれを適用してみましょう。
共有する頂点を $\mathrm{O}$ とし、3つの辺を直線 $\mathrm{OA}$, 直線 $\mathrm{OB}$, 直線 $\mathrm{ON}$ にとります。
しかし、点 $\mathrm{M}$ は直線 $\mathrm{OB}$ 上にはありません。この公式は「3つの辺が対応していること」が前提となるため、今回の「底面と高さに分ける」アプローチが最も確実で汎用性が高いです。
公式が使える形に変形することも可能ですが、かえって時間がかかるため、無理に公式に当てはめようとせず図形的意味（底面積 $\times$ 高さ）に忠実に解くのがベストです。

受験生のつまずきポイント

点 $\mathrm{N}$ が平面上にある条件の誤用

平面上にある条件として「係数の和が $1$」と丸暗記している生徒が非常に多いです。
$\Vec{ON} = \alpha\Vec{OA} + \beta\Vec{OB} + \gamma\Vec{OC}$ において、係数和 $\alpha+\beta+\gamma=1$ が使えるのは、点 $\mathrm{N}$ が平面 $\mathrm{ABC}$ 上にあるときだけです。
今回のように「平面 $\mathrm{OAC}$」上にある場合は、原点 $\mathrm{O}$ を含む平面であるため、「はみ出しているベクトル（$\Vec{OB}$）の係数が $0$」とするのが正解です。

内分の比率の逆転ミス

$\Vec{OM} = \dfrac{\Vec{OB} + 5\Vec{ON}}{6}$ の式を見て、「点 $\mathrm{M}$ は線分 $\mathrm{BN}$ を $1:5$ に内分する」と答えてしまうミスが頻発します。クロスの法則を思い出してください。$\Vec{ON}$ の係数が $5$ であるということは、$\mathrm{M}$ は $\mathrm{B}$ 側から見て $5$、$\mathrm{N}$ 側から見て $1$ の位置、つまり $5:1$ に内分しています。この逆転ミスで体積が $\frac{5}{6}$ 倍になってしまう痛い失点を防ぎましょう。

アクティブリコール（復習テスト）

今日の学習内容を思い出すことで、記憶を定着させましょう。質問をクリックして答えを確認してください。

Q1. 空間において、点 $\mathrm{P}$ が「直線 $\mathrm{AB}$ 上」にあるためのベクトル方程式はどう表されますか？
実数 $s$ を用いて、$\Vec{OP} = (1-s)\Vec{OA} + s\Vec{OB}$ と表されます。（係数の和が1になるのが特徴です）。
Q2. 空間において、点 $\mathrm{P}$ が「平面 $\mathrm{OAB}$ 上」にあるためのベクトル方程式の条件は何ですか？（基準を $\Vec{OA}, \Vec{OB}, \Vec{OC}$ とした場合）
$\Vec{OP}$ を $\Vec{OA}, \Vec{OB}, \Vec{OC}$ の1次結合で表したとき、$\Vec{OC}$ の係数が $0$ になります。原点 $\mathrm{O}$ を含む平面上の点は、その平面を張る2つのベクトル（今回は $\Vec{OA}, \Vec{OB}$）だけで表せるためです。
Q3. ベクトル方程式 $\Vec{OP} = \dfrac{3\Vec{OA} + 4\Vec{OB}}{7}$ が与えられたとき、点 $\mathrm{P}$ は線分 $\mathrm{AB}$ を何対何に内分する点ですか？
「$4 : 3$」に内分する点です。ベクトルの内分公式では、分母の比率がクロスして掛けられるため、$\Vec{OA}$ の係数が $3$、$\Vec{OB}$ の係数が $4$ ならば、比率は逆に $4:3$ となります。
